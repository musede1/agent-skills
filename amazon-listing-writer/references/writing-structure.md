# 写作结构模板

## 标题（Title）结构

### 推荐结构

```
[品类词] + [高精准修饰锚点（优先题材形状词）] + [材质词（可选）] + [表面工艺词（可选）] + [场景词（可选）] + [礼品场景词（可选）]
```

### 长度要求

| 指标 | 要求 |
|------|------|
| 硬上限 | ≤190字符（含空格） |
| 目标区间 | 150-190字符（含空格） |
| 前段必覆盖 | 前70-90字符包含品类词+高精准修饰锚点 |

### 核心原则

- 可读性优先
- 不堆砌
- 不引入未确认事实
- 不使用夸大词
- **不写尺寸**

### 示例

**S1模式（自用为主）**：
```
Couple Statue Resin Figurine Abstract Sculpture for Home Decor Living Room Bedroom Mantel Shelf Tabletop Display Modern Art
```

**S2模式（礼品意图）**：
```
Couple Statue Resin Figurine for Home Decor Anniversary Wedding Gift for Husband Wife Lovers Living Room Bedroom Mantel Display
```

## 五点（Bullets）结构

### 推荐5条主题布局

| 条目 | 主题 | 核心内容 |
|------|------|----------|
| Bullet 1 | 主题与情绪价值 | 题材形状词 + 受众词 |
| Bullet 2 | 尺寸与摆放 | 尺寸规格词（必写） + 摆放位置词/空间场景词 |
| Bullet 3 | 材质与表面工艺 | 材质词 + 表面工艺词 |
| Bullet 4 | 稳定性/组合数量/空间适配 | 组合数量词 + 场景词/风格词 |
| Bullet 5 | 礼品场景与包装透明 | 礼品场景词/节日场合词 + 包装真实描述 |

### 长度要求

| 指标 | 要求 |
|------|------|
| 每条目标区间 | 170-230字符（含空格） |
| 硬上限 | ≤250字符（含空格） |

### 真值覆盖要求

| 维度 | 要求 |
|------|------|
| 单条底线 | ≥1个可核验真值点 |
| 全局配额 | 5条累计≥8个不同真值点 |

### 模板示例

**Bullet 1 - 主题与情绪价值**：
```
SYMBOLIC COUPLE DESIGN - This [题材形状词] figurine depicts [具体描述]. Features [特征描述]. Perfect [情绪价值] decor for [受众词] celebrating [场景/意义].
```

**Bullet 2 - 尺寸与摆放**：
```
PERFECT SIZE FOR DISPLAY - Measures approximately [高] x [宽] x [深] inches ([cm换算]). Ideal dimensions for display on [摆放位置词列举], without overwhelming the space.
```

**Bullet 3 - 材质与表面工艺**：
```
[材质词] CONSTRUCTION - Crafted from [具体材质] with a [表面工艺词] finish. [耐久性描述]. Easy to clean with [清洁方式]. Suitable for [使用场景限制].
```

**Bullet 4 - 稳定性/组合数量**：
```
[稳定性特征] DESIGN - Features [底座/结构描述] for secure placement. [组合数量] included. The [设计特点] ensures [安全/稳定效果].
```

**Bullet 5 - 礼品场景与包装**：
```
[礼品定位] FOR LOVED ONES - [包装描述（有什么写什么）]. Ideal gift choice for [节日场合词列举]. [如有礼盒则写 Comes in elegant gift packaging].
```
注意：无礼盒时不写"不含礼盒"，避免负面暗示影响转化。

## 后台搜索词（Search Terms）结构

### 格式要求

- 英文小写
- 空格分隔
- 无标点
- 长度区间：230-250 bytes（尽量写满，提高权重覆盖面）

### 禁止内容

- 品牌词/IP词
- 竞品品牌
- 夸大词（best/top）

### 覆盖优先级

1. 品类词同义词（statue/figurine/sculpture）
2. 空间场景词
3. 摆放位置词
4. 受众词变体
5. 礼品场景词
6. 节日场合词
7. 风格词

### 示例

```
couple figurine statue sculpture abstract modern resin decor anniversary wedding gift husband wife lovers newlywed housewarming living room bedroom shelf mantel tabletop entryway office desk
```

## 长描述（Description）结构

详见 references/description-rules.md

### 6段式结构概览

| 段落 | 主题 | 占比 | 字符数 |
|------|------|------|--------|
| 第1段 | 开篇（情感钩子 + 产品定位） | 15% | 180-285 |
| 第2段 | 核心价值（题材/设计/工艺亮点） | 25% | 300-475 |
| 第3段 | 规格信息（尺寸/材质/表面工艺） | 20% | 240-380 |
| 第4段 | 场景描绘（摆放场景/空间适配） | 20% | 240-380 |
| 第5段 | 礼品价值（适合场合/受众） | 15% | 180-285 |
| 第6段 | 收尾（行动号召） | 5% | 60-95 |

### 长度要求

| 指标 | 要求 |
|------|------|
| 最低长度 | 1200字符（含空格） |
| 最高长度 | 1900字符（含空格） |

### 与五点差异化

- 五点：功能导向、简洁直接、分点罗列
- 长描述：叙事性、情感共鸣、段落流畅

**禁止**：直接复制五点内容

## 多站点适配

### 美国站（US）

- 语言：en
- 尺寸：优先inches，括号内标注cm
- 长度限制：标题≤200字符（安全区190）

### 欧洲站（UK/DE/FR/IT/ES）

- 语言：对应站点语言
- 尺寸：优先cm，括号内标注inches
- 注意本地化表达习惯

### 加拿大站（CA）

- 语言：en（部分支持fr）
- 尺寸：同US
- 关键词可参考US站

## 写作原则总结

1. **精准优先**：只写已确认的真值
2. **可读性优先**：自然流畅，不堆砌
3. **转化优先**：覆盖买家关心的问题
4. **合规优先**：不夸大、不承诺、不侵权
5. **Rufus友好**：可问可答式覆盖

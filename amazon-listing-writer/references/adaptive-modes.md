# 自适应模式详解（Adaptive Modes）

## S0 自适应核心词选择与落位

**默认采用S0**：根据产品特性与意图自动选择S1或S2

---

## 模式定义

### S1 极致精准模式

**触发条件**：默认模式，未检测到礼品意图时使用

**标题策略**：
- 优先选择 [品类词 + 高精准修饰锚点]
- 必要时可加材质词/表面工艺词（仅已确认且不损可读性）

**核心短语**：品类词为主，修饰锚点强化产品特征

---

### S2 精准+礼品平衡模式

**触发条件**（满足任一即升级）：
1. P0中出现：节日场合词 / 礼品场景词 / 明确礼品受众
2. P2 CRD辅助信号：`gift_intent_signal.recommendation.suggest_s2_mode = true`（辅助依据，非唯一依据）
3. P1意图词自动提炼规则筛出强证据的礼品场景词/节日场合词/受众词（且通过同框过滤）

**标题策略**：
- 在不牺牲可读性前提下，允许加入"带对象/人群的礼品场景词"
- 示例：wedding gift for couple
- 仍必须由产品词（品类词）锚定

---

### S3 增长扩展模式

**触发条件**：仅在用户明确指令时启用

**标题策略**：
- 允许更泛礼品词进入标题后半段
- 同步强化预期管理（尺寸/包含物等）

---

## 标题核心词规则（必须遵守）

### 品类词必须包含

标题必须包含品类词，避免只写礼品词而不说明"卖什么"

### 核心短语数量

- 最少：1个
- 最多：2个
- 门槛：高精准度

### 禁止写入标题的内容

| 禁止项 | 原因 |
|--------|------|
| 尺寸规格词 | 尺寸只能写入五点 |
| 颜色词（前70-90字符） | 父体标题前段禁入 |
| 价格档次词（前70-90字符） | 父体标题前段禁入 |

### 标题长度规则

| 指标 | 要求 |
|------|------|
| 硬上限 | ≤190字符（含空格） |
| 目标区间 | 150-190字符（含空格） |

### 前段优先规则

前70-90字符必须覆盖：
- 品类词
- 至少1个高精准修饰锚点

**修饰锚点优先级**（依次使用）：
1. 题材形状词
2. 核心特征锚点
3. 功能属性词
4. 材质词
5. 风格词（用户提供或P1强证据提炼且图片低争议匹配）
6. 受众词（如触发S2时）

---

## 五点必埋词规则

### 数量要求

- 最少：2个
- 最多：5个
- 方式：自然分散嵌入，不堆砌

### 优先类型（按顺序）

1. 题材形状词
2. 受众词
3. 礼品场景词（仅S2时或P1强证据提炼成立时）
4. 摆放位置词
5. 空间场景词
6. 功能词
7. 表面工艺词
8. 材质词
9. 尺寸规格词
10. 节日场合词

### 单条承载限制

每条Bullet承载"核心短语"不超过2个

---

## 五点长度与真值覆盖（方案C）

### 长度要求

| 指标 | 要求 |
|------|------|
| 每条目标区间 | 170-230字符（含空格） |
| 硬上限 | ≤250字符（含空格） |

### 真值覆盖要求

| 维度 | 要求 |
|------|------|
| 单条真值点底线 | ≥1个可核验真值点 |
| 全局真值覆盖配额 | 5条累计≥8个不同真值点 |

### 安全扩写模块

当单条真值点不足以写满目标长度时，只允许使用以下内容补足字数：

| 类型 | 说明 |
|------|------|
| 空间场景词 | 禁止自动推断bathroom/outdoor等强绑定场景 |
| 摆放位置词 | shelf/mantel/tabletop等 |
| 品类词同义覆盖 | wall decor/wall art/wall sculpture |
| 风格词 | 仅限P1强证据提炼且图片低争议匹配 |
| 礼品场景词/节日场合词/受众词 | 当触发S2或P1强证据成立时可用 |
| 基于P0真值的低风险搭配建议 | 如"有开口可插花"→ faux/dried/greenery |

**禁止**：引入新承诺、夸大词、反复写包装以凑字数

### 预期管理/包装透明

仅在必要位置出现一次即可，不得作为扩写灌水模块反复堆叠

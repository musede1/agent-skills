# 关键词类型命名体系

## 关键词分类定义

用于清洗、归类与落位

| 类型 | 英文名 | 示例 |
|------|--------|------|
| 品类词 | Category | statue, figurine, sculpture, vase, wall decor, garden statue |
| 风格词 | Style | modern, minimalist, farmhouse, boho, rustic, contemporary |
| 空间场景词 | Space/Room | living room, bedroom, entryway, office, patio |
| 材质词 | Material | resin, ceramic, metal, wood, polyresin |
| 颜色词 | Color | white, black, gold, brown, bronze, silver |
| 功能词 | Function | decorative, indoor, outdoor, hangable, watertight |
| 题材形状词 | Theme/Shape | couple, angel, abstract, animal, religious |
| 受众词 | Audience | for couples, for newlyweds, for husband and wife |
| 尺寸规格词 | Size | 6 inch, 15cm, tall, wide, opening diameter |
| 节日场合词 | Occasion | Valentine's Day, anniversary, Mother's Day, Christmas |
| 价格档次词 | Price Tier | premium, affordable（慎用） |
| 组合数量词 | Quantity | single, set of 2, pair |
| 摆放位置词 | Placement | mantel, shelf, tabletop, desk, garden |
| 表面工艺词 | Finish | glossy, matte, faux stone, distressed, glazed |
| 礼品场景词 | Gift Scene | wedding gift, engagement gift, housewarming gift |
| 品牌词 | Brand | 仅限用户自有品牌 |

## 清洗规则（必须执行）

### 严禁使用

| 类型 | 示例 |
|------|------|
| 竞品品牌词 | 任何非用户自有品牌 |
| IP词/角色名 | Disney, Marvel角色等 |
| 受保护形象 | 版权角色、商标形象 |

### 避免使用

| 类型 | 示例 |
|------|------|
| 夸大营销词 | best, #1, free, top quality, world's best |
| 不相关引流词 | 会拉低转化的无关词 |
| 主观夸张风格词 | luxury, premium, elegant（除非有明确定位支撑） |

## 关键词落位指南

### 标题落位

| 优先级 | 词类型 | 说明 |
|--------|--------|------|
| 必须 | 品类词 | 标题必须包含，明确卖什么 |
| 优先 | 题材形状词 | 作为高精准修饰锚点 |
| 可选 | 材质词 | 已确认且不损可读性时 |
| 可选 | 空间场景词 | 需满足自然排名门槛 |
| 条件 | 礼品场景词 | 仅S2模式 |
| 条件 | 风格词 | 仅按例外规则入标题 |
| 禁止 | 尺寸规格词 | 只能写入五点 |
| 禁止前段 | 颜色词、价格档次词 | 只能在标题后段 |

### 五点落位

| 优先级 | 词类型 |
|--------|--------|
| 必须 | 尺寸规格词（含单位与approx换算） |
| 高 | 题材形状词、受众词 |
| 中 | 材质词、表面工艺词、功能词 |
| 中 | 摆放位置词、空间场景词 |
| 条件 | 礼品场景词、节日场合词（S2或P1强证据时） |
| 低 | 组合数量词、颜色词 |

## Rufus友好覆盖

在Bullets中用"可问可答"方式写清（仅限真值）：

| 内容 | 覆盖方式 |
|------|----------|
| 尺寸规格 | 仅在五点出现，含单位与approx换算 |
| 材质与表面工艺 | 如 glossy/matte |
| 组合数量与包含物 | 如含底座、含挂钩、comes in gift box |
| 适用摆放位置/空间场景 | 允许自动生成，但不做强承诺 |
| 护理与使用边界 | 仅在已确认时写，避免强承诺 |

### 同义表达自然覆盖

- figurine / statue / sculpture
- gift for couple / newlywed / husband and wife
- mantel / shelf / tabletop
- living room / bedroom / entryway

# 输入要求详解（Input Requirements）

## 数据优先级

越靠前权重越高，必须严格遵守：

| 优先级 | 类型 | 说明 |
|--------|------|------|
| P0 | 产品真值 | 最高权重，必须准确，严禁编造 |
| P1 | 关键词数据 | 高权重，含自动提炼规则（intent_extractor.py 4层锚点） |
| P2 | CRD辅助验证 | 辅助权重，竞品评论分析结论（Phase A输出） |
| P3 | 补充信息 | 包装细节、禁用词、品牌调性 |

---

## P0 产品真值字段（严禁编造）

| 字段 | 说明 | 来源要求 |
|------|------|----------|
| 站点 | US/CA/MX/UK/DE/FR/IT/ES等 | 用户指定或默认US |
| 品类词 | 雕塑/雕像/花瓶/墙饰/户外雕塑 | 用户提供 |
| 材质词 | 树脂/陶瓷/金属/木等 | 用户确认 |
| 功能词 | 纯装饰/可盛水/可悬挂/可户外 | 仅写已确认 |
| 题材形状词 | 情侣/天使/动物/抽象/人物姿态 | 用户提供 |
| 核心特征锚点 | 1-3个最关键差异点 | 必须真实 |
| 风格词 | 现代/极简/农舍/波西米亚/复古 | 用户提供或P1强证据提炼 |
| 表面工艺词 | 哑光/亮面/做旧/仿石/电镀/釉面 | 仅写已确认 |
| 尺寸规格词 | 高/宽/深、口径、重量 | 必须明确单位，自动换算 |
| 颜色词 | 主色/辅色 | 仅写已确认 |
| 价格档次词 | 高端/中端/走量 | 影响语气，未给则不写 |
| 组合数量词 | 单个/一对/套装 | 必须真实 |
| 受众词 | 情侣/夫妻/伴侣/新婚 | 用户提供或P1提炼 |
| 节日场合词 | 情人节/纪念日/婚礼/乔迁 | 用户提供或P1提炼 |
| 礼品场景词 | wedding gift/anniversary gift | 用户提供或P1提炼 |
| 品牌词 | 仅限用户自有品牌 | 严禁竞品品牌/IP |

### 尺寸单位换算规则

若只给一种单位，自动换算另一种并标注"approx."

## P1 关键词数据

### 用户主动提供核心关键词清单

作为"优先核心词池"，必须优先挑选并埋入：
- 标题：1-2个（高精准度门槛）
- 五点：2-5个（自然分散嵌入）

### 用户未提供核心关键词清单

从用户上传CSV/表格/竞品关键词库中自动提炼，埋入规则同上。

## P2 CRD辅助验证（Competitor Review Dataset）

来源：Phase A输出的`review_analysis.json`

**三大辅助角色**：
- 角色1（预期管理顾问）：`negative_pain_points` + `size_material_concerns` → 影响五点预期管理措辞优先级
- 角色2（S2触发信号）：`gift_intent_signal` → S2模式触发的辅助判断依据（非唯一依据）
- 角色3（表达参考库）：`buyer_vocabulary` + `positive_selling_points` → 五点和Description的语言风格参考

**使用限制**：
- CRD结论为辅助参考，不替代P0真值和P1关键词库
- 严禁将竞品评论中的事实当作我方产品事实（核心合规原则）
- buyer_vocabulary仅供语言风格参考，不参与L1-L4锚点词配置

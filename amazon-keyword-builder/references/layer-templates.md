# Layer模板详解

## LayerA（仅当L1≠无）

### A0基础覆盖（必须）

```
模板：+L1 CT变体
数量：2-5条
```

### A0特征强化（条件）

```
模板：+L1 +外观特征锚点 CT变体
条件：外观特征锚点在关键词库有证据
```

### A1功能结构（条件）

```
模板：+L1 +功能结构锚点 CT变体
条件：关键词库有功能/结构高频词
```

### A1场景礼品（条件）

```
模板：+L1 +礼品场景锚点 CT变体
条件：关键词库有礼品场景证据
```

---

## LayerB（L2满足准入时）

```
模板：+核心特征锚点 +L2 CT变体
标注：L2a备注"L2a-直接父类"；L2b备注"L2b-超泛父类+更泛/易跑偏"
```

---

## 颜色组

**来源**：只从关键词库提取

### 模板（L1≠无）

```
强：+颜色 +核心特征锚点 +L1 CT变体
弱：+颜色 +L1 CT变体
```

### 模板（L1=无）

```
强：+颜色 +材质/工艺 CT变体
弱：+颜色 +材质 CT变体
```

**备注必须写**："颜色来源=关键词库;运营需根据实际子体颜色选择投放"

---

## LayerC多维候选池

### 维度配置

| 维度 | 阈值 | 保底 |
|------|------|------|
| 功能 | 宽松 | ✅ |
| 材质 | 宽松 | ✅ |
| 摆放位置 | 宽松 | ✅ |
| 工艺 | 宽松 | ❌ |
| 表面工艺光感 | 宽松 | ❌ |
| 风格 | 严格 | ❌ |
| 空间场景 | 严格 | ❌ |

### L1禁用规则

LayerC关键词中**禁止出现L1/L2/题材词**

### 模板判别

**强证据+非L1共现存在**：
```
+维度锚点 CT变体
```

**否则**：
```
+维度锚点 +第二锚点 CT变体
```

### 保底维度

必须输出，标注"保底输出;证据弱;低预算低出价测试"

---

## LayerC-品牌

### 控边界规则

必须同时满足：
1. 锁定L1/L2/核心特征锚点（至少一个）
2. 带CT

### 模板

```
+品牌词 +L1或L2或核心特征 CT变体
风险标记：更泛/高CPC风险/需复核
备注：品牌词仅用于广告专组;不得混入普通组;禁止写入Listing
```

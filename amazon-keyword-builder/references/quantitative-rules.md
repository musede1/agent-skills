# 量化判定规则

## 1. 强证据题材词判定（L1候选）

### 指标定义

| 指标 | 定义 |
|------|------|
| F | 出现频次 |
| U | 覆盖关键词数 |
| SV | 搜索量加总 |
| R | 最佳自然排名 |

### 阈值

```
F ≥ 2, U ≥ 2, SV ≥ 500, R ≤ 30
```

### 判定

满足 ≥2 个条件 → 强证据

---

## 2. LayerC强证据维度词判定

### 统计口径

非L1子集 ∩ 类目子集（含CT的关键词）

### 宽松阈值（功能/材质/工艺/表面工艺光感/摆放位置）

```
F ≥ 2, U ≥ 2, SV ≥ 500, R ≤ 30
满足 ≥2 条 → 强证据
```

### 严格阈值（风格/空间场景）

```
F ≥ 3, U ≥ 3, SV ≥ 800, R ≤ 25
满足 ≥2 条 → 强证据
```

---

## 3. 非L1共现承接判定

满足以下全部条件：
1. 含维度锚点词
2. 含任一CT
3. 不含L1

→ 非L1共现承接=存在

---

## 4. L2准入规则

### L2a准入（严格）

1. L1+L2同条共现存在
2. 在L1子集中满足强证据

### L2b准入

1. 全库满足强证据
2. L2+CT共现存在
3. 不与L2a相同

---

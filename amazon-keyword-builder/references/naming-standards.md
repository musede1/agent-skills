# 分组命名标准

**禁止自由命名，必须使用以下固定名称**

---

## Sheet1 分组名

### LayerA

| 分组名 | 说明 |
|--------|------|
| LayerA_A0基础覆盖 | +L1 CT |
| LayerA_A0特征强化 | +L1 +外观特征 CT |
| LayerA_A1功能结构 | +L1 +功能结构 CT |
| LayerA_A1场景礼品 | +L1 +礼品场景 CT |
| LayerA_L1备选-{题材} | 备选题材，如 LayerA_L1备选-hippo |

### LayerB

| 分组名 | 说明 |
|--------|------|
| LayerB_L2上位类扩量 | +核心特征 +L2 CT |

### 颜色组

| 分组名 | 说明 |
|--------|------|
| 颜色组 | 固定名称 |

### LayerC

| 分组名 | 说明 |
|--------|------|
| LayerC-功能 | 保底 |
| LayerC-材质 | 保底 |
| LayerC-摆放位置 | 保底 |
| LayerC-工艺 | 非保底 |
| LayerC-表面工艺光感 | 非保底 |
| LayerC-风格 | 非保底 |
| LayerC-空间场景 | 非保底 |
| LayerC-品牌 | 品牌词专组 |

### 其他

| 分组名 | 说明 |
|--------|------|
| 拼写组 | 拼写错误词 |

---

## Sheet2 分组名

| 分组名 | 说明 |
|--------|------|
| Exact-E1核心成交 | L1+CT核心 |
| Exact-E2高意图长尾 | L1+CT+修饰 |
| Exact-E3颜色变体 | 带颜色 |
| Exact-E4防御扩量 | L2/特征防御 |
| 品牌词-Exact | 品牌词 |

---

## 错误示例

```
❌ LayerA_狗题材      → ✅ LayerA_A0基础覆盖
❌ 颜色词组           → ✅ 颜色组
❌ Layer C 功能       → ✅ LayerC-功能
❌ 品牌词             → ✅ LayerC-品牌 或 品牌词-Exact
```

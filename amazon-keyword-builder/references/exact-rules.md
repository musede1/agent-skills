# Exact精准词规则

## 目标

新品期10-25条高意图Exact

---

## 挑选规则

### L1 ≠ 无

**优先结构**：
```
a) L1 + CT + 修饰词（外观/功能/材质/工艺）
b) L1 + CT
c) 外观特征 + L1
```

**L2融合**（防御扩量）：
```
a) 核心特征 + L2 + CT
b) L2 + CT（更泛）
c) 颜色 + L2 + CT（低优先级）
```

### L1 = 无

```
L2 + CT + 材质/工艺/颜色（控边界）
或
功能/材质/核心特征 + CT
```

---

## 品牌词-Exact

- 数量：1-10条
- 只输出Exact，不做Phrase
- 风险标记：更泛/高CPC风险/需复核
- 备注：品牌词仅用于广告专组;不得混入普通组;禁止写入Listing

## 去重规则

### 组内

保留"更短、锚点更强、证据更强"的代表词

### 跨组

锚点类型不同必须保留

### CT归一

同一词内只保留一个CT，同一题材最多2-5个CT变体

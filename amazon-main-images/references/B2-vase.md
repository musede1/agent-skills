## 一、适用范围

- 仅适用于花瓶类产品(B类)

- 不适用于 A / C / D 类

## 二、核心原则

1. 花瓶存在两种视觉状态:**空瓶**(展示工艺) / **满花**(展示功能)。
2. 空瓶 vs 满花的分配不是临时决定,必须由 I2 心智方案在阶段1就锁定。
3. 花材是花瓶类的功能表达,不是普通道具,不计入风格包"道具方向"约束。
4. 同一花材全片最多出现 2 次,第 3 次强制更换。

---

## 三、花瓶呈现模式(B 类唯一模式)

B 类只保留一种呈现模式:**纯生活型**(执行 H-B1)。

> 为什么砍到只剩一种:花瓶的核心商业证据是"插上花好看",空瓶证明不了功能。空瓶图只在 Image_2 尺寸图名额内允许存在(执行 H-B2)。原"纯工艺型/均衡型"在亚马逊主图副图集里转化数据明显劣于纯生活型,故废除。

### I2 推导规则(B 类)

- I2 输出 3 个候选心智方案时,3 个方案的**主导心智不同**(差异在心智、不在模式)
- 每个候选方案必须显式标注:`花瓶呈现模式:纯生活型`
- 候选方案的职责权重必须满足 H-B2 + H-B3

## 四、花瓶专属职责(D1 交互组扩展)

> R05 的"商业动作"在视频里是连续的镜头链,在主图里改成**单张图捕捉的动作瞬间**。
> 这些职责追加到 D1 交互组,B 类专用,不影响其他品类。

| 职责名称 | 单图呈现 | 手部 | 通常对应模式 |
|---|---|---|---|
| **双手捧瓶** | 双手从两侧捧住瓶身,展示体量与质感 | 双手 | 任意 |
| **转瓶展示** | 单手扶瓶身侧面,呈现"转动中"的瞬间 | 单手 | 有明确独家理由时启用 |
| **瓶口俯览** | 手轻扶瓶身,俯角拍摄展示瓶口造型 | 单手 | 有明确独家理由时启用 |
| **插花瞬间** | 单手持一支花茎,正要送入瓶口的瞬间 | 单手 | 纯生活型/均衡型 |
| **整束入瓶** | 双手捧一束花,正要放入宽口的瞬间 | 双手 | 纯生活型 |
| **注水瞬间** | 单手持壶/杯,水线正在落入瓶口的瞬间 | 单手 | 纯生活型/均衡型 |
| **落位归置** | 双手将瓶放至台面,刚刚松手的瞬间 | 双手 | 任意 |
| **礼品递送** | 双手捧瓶向镜头递出 | 双手 | 礼品转化心智 |

### 职责选用规则

- 全片至少 1 张交互职责(继承 D1 总规则)
- 同一种交互职责全片只用 1 次
- "注水瞬间"和"插花瞬间"不可同时出现(避免动作语义冲突)
- 交互职责的动作必须与该图的空瓶/满花状态自洽:
  - 注水瞬间 → 该图必为空瓶或半空(瓶里不能已经有花)
  - 插花瞬间 → 该图为半满状态(瓶里已有部分花,正在加新的)
  - 整束入瓶 → 该图为空瓶状态(瓶里还没有花)
  - 落位归置 → 空瓶或满花皆可
  - 双手捧瓶/转瓶/瓶口俯览 → 通常空瓶
  - 礼品递送 → 通常空瓶或精致小束

### 5.5 花材选色由心智决定(非铁律,判断指引)

花材选色不是"克制就对、鲜艳就错"的一刀切问题,而是由 I2 锁定的心智方案决定。不同心智方向对应不同的花材色策略:

| 心智方向 | 花材选色策略 | 例子 |
|---|---|---|
| 艺术品收藏向 | 克制色为主(白/米白/淡绿/淡粉),让产品本体当主角 | 白芍药、米白洋桔梗、淡绿尤加利 |
| 礼品转化 / 居家活力 / 节庆氛围 | 鲜艳色合法(红/紫/橙/明黄),让画面氛围当主角 | 红玫瑰、紫绣球、橙郁金香、向日葵 |
| 展示百搭实用 | 跨色相变化,9 张图允许混用克制色和鲜艳色,证明产品能 hold 各种花 | 白芍药 + 红玫瑰 + 紫绣球 + 米白洋桔梗等组合 |

**唯一的硬约束(花材色 vs 产品本体色)**:

同一片 9 图内,**禁止花材色与产品本体主色完全相同**。例如:
- 白瓶配纯白花 → 花瓶轮廓被花稀释,产品消失在花里
- 黄瓶配同色黄花 → 黄上加黄,瓶身渐变层次被花拉平
- 蓝瓶配同色蓝花 → 同上

这条硬约束的原因是:产品的视觉跳点要靠"产品本体的色相 + 饱和度对比"建立,如果花材和产品同色相,跳点会从"产品独占"退化为"瓶+花共同承担",产品本体的特色被花稀释。

**同色相但不同明度允许**(如黄瓶配米白花、白瓶配淡灰绿花),因为明度差能保住产品轮廓。

**判断顺序**:
1. 先看 I2 锁定的心智方案 → 决定花材色策略(克制 / 鲜艳 / 跨色相)
2. 再检查花材色 vs 产品本体色 → 触发硬约束就换花
3. 最后填写 §五 花材分配总表

## 六、空瓶 vs 满花的构图策略

### 空瓶状态

- 构图焦点:瓶身轮廓、纹理、釉色、瓶口造型
- 瓶口必须完整可见,不被遮挡
- 光线优先打透瓶体或斜打表面,展示材质质感
- 适用职责:借景型置景 / 尺寸图 / 部分场景图

### 满花状态

- 构图焦点:花材+瓶体的整体组合
- 允许花头轻微遮挡瓶口边缘(饱满的自然结果)
- 但瓶身中下段必须清晰可见,不能被花材完全遮挡
- 光线兼顾花材色彩与瓶体质感
- 适用职责:场景类多数 / 交互类(插花/整束/注水)

### 半满状态

- 仅用于"插花瞬间"职责
- 瓶里已有少量花,手部正在加新的
- 构图焦点同时落在瓶口、已有花材、手部动作三者

## 七、花材体量基准(按瓶口宽度)

> 主图体系下,花材描述聚焦视觉饱满度,不写支数。
> 但内部判断花量要参考瓶口宽度。

| 瓶口宽度 | 内部判断的花量参考 | 目标视觉 |
|---|---|---|
| 窄口(≤5cm) | 3–5 主花 + 1–2 配花 | 花茎紧密聚拢,花头自然散开高于瓶口 |
| 中口(5–10cm) | 5–8 主花 + 2–3 配花 | 花束饱满充盈,花头覆盖瓶口大部分 |
| 宽口(≥10cm) | 8–12 主花 + 3–5 配花 | 花束丰盛溢出,花头几乎完全覆盖瓶口 |

### 视觉铁律(仅对满花/半满图生效)

> 空瓶图不受本节约束,空瓶图执行第六节"空瓶状态构图策略"。

- 满花图禁止"瓶口大面积空置只插 1–2 支孤花"的画面
- 花头总面积 ≥ 瓶口面积 70%
- 花束高度 ≥ 瓶身高度 0.6 倍
- 花头允许轻微遮挡瓶口边缘(饱满自然结果)
- 特写景别可只拍局部花材,不受此限

## 八、瓶口遮挡关系(按景别)

| 景别 | 瓶口可见度要求 |
|---|---|
| 全景 / 中景 | 花材可覆盖瓶口,瓶身轮廓清晰可辨 |
| 近景 | 瓶口边缘至少露出 30%,让买家看到瓶口造型 |
| 特写 | 拍瓶口 → 无花;拍花材 → 瓶口作为背景部分可见 |
| 交互手部图 | 瓶口必须清晰可见,买家需看到手部动作如何作用于瓶口 |

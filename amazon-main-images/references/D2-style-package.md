# D2-style-package.md 风格包生成规范

> 风格包是 9 张图视觉一致性的唯一保证。
> 9 张图像不像一套,100% 取决于风格包写得好不好。

---

## 一、核心地位

风格包扮演三个关键角色:
1. 9 张图一致性的唯一保证
2. Nano Banana 能听懂的翻译层(把抽象风格名翻译成具体参数)
3. 每张图自检的标尺

---

## 二、生成规范

- 每次基于锁定风格 + 当前产品信号现场生成,不套用预设模板
- 完备:覆盖所有视觉元素类别
- 具体:每项可被 Nano Banana 直接执行
- 自洽:参数互不打架

---

## 三、风格包必含字段

```
== 主色盘 ==
墙面/背景:[具体色族]
台面/落点:[具体色族]
软装底色:[具体色族]

== 60-30-10 配色判断(隐式约束,作为审核标准,不写进 prompt)==

第一步:判断产品色彩强度
  · 产品色彩较强(高饱和 / 渐变色 / 强材质 / 撞色)→ 走方案 A
  · 产品色彩较弱(纯白 / 透明无色 / 单色低饱和 / 哑光素色)→ 走方案 B

方案 A:产品独占 10%(强色产品默认走这个)
  · 60% 主色:墙 + 大面积软装(沙发/窗帘/床品/地毯),低饱和暖中性基调,允许暖白/米白/暖灰内部明度变化
  · 30% 辅色:家具 + 台面 + 中等织物,低饱和但允许跨色相(暖橡木 + 冷灰亚麻 + 深绿绿植 + 灰蓝木门等组合都合法,只要明度协调互不打架)
  · 10% 跳点:**产品本身**,靠饱和度对比取胜,不依赖"周围同色衬托"
  · **禁止再加任何呼应色锚点**(再加会稀释产品的跳点地位)
  · 道具按真实居家空间自然出现,不指定鲜艳颜色,避免色彩抢戏

  **关键澄清**:60-30-10 是**面积比例规则**,不是色相数量规则。"低饱和"≠"同色族",60% 和 30% 之间允许暖灰、冷灰、暖木、深绿等不同色相共存,只要全部保持低饱和,产品的高饱和跳点地位就不会被破坏。

  **常见误区**:
  · ❌ 把"低饱和"理解成"必须全部米白" → 出图变成"九张米白循环",画面无层次
  · ❌ 因为"统一感"而排除灰沙发/灰蓝门/沙绿墙/深绿绿植 → 这些都是合法的 30% 辅色成员
  · ✅ 正确做法:在 60% 主色保持暖白/米白基调的前提下,30% 辅色大胆引入跨色相低饱和元素(灰、绿、蓝、深木),给画面建立明度锚和冷暖对冲

方案 B:产品 + 呼应物共同构成 10%(弱色产品才走这个)
  · 60% 主色:同方案 A
  · 30% 辅色:同方案 A
  · 10% 跳点:产品 5% + 1 个呼应物 3-5%
  · 呼应物色相要求:与产品形成柔和对比或同色系深一度,**不能选鲜艳跳色**(紫/橙/红等)
  · 呼应物载体:每张图选**这个房间最自然的物件**承载(书房放书 / 厨房放砧板 / 卧室放钟 / 玄关放小盘),载体可以变化但色相必须统一
  · **禁止 9 张图用同一种载体**(防止"九本书"症)

铁律:
  · 60-30-10 是审核标准,不写进 prompt 字眼
  · 写 prompt 时只描述"真实居家空间",配色由风格设定 + 主色盘隐式产生
  · 审核 prompt 时反向检查:主色 60% 是否统一 / 辅色 30% 是否统一 / 跳点 10% 是否干净不打架
  · 产品色彩强度判断错误 → 重写
  · 错把强色产品塞呼应物 → 立即删除呼应物
  · 错把弱色产品做成"产品独占" → 加 1 个呼应物

色彩强度判断常见错误:
  · ❌ 把"白色 + 金边"陶瓷判定为弱色(其实金边就是跳点,走方案 A)
  · ❌ 把"深蓝玻璃"判定为弱色(深色高饱和也是强色,走方案 A)
  · ❌ 把"乳白渐变薄荷绿玻璃"判定为弱色(渐变色 = 强色,走方案 A)
  · ✅ 真正的弱色产品例:全白哑光花瓶 / 全透明无色玻璃 / 全米色亚麻收纳篮

== 全片核心落点 ==(高优先级新增字段)
落点材质类型:[具体材质,如"浅暖橡木板"或"白大理石板"]
落点选择原则:存在感低,质感高(避免抢戏)
应用范围:置景类全部使用同一落点
**场景图豁免**:场景图的落点跟随各自空间的真实家具自然变化(客厅茶几 / 卧室梳妆台 / 餐厅餐桌等),无需统一,但 60-30-10 主辅色铁律仍然生效
例外:Image_2 用独立标准底(陶板/石板) / 纯特写图无落点

== 色温与饱和度 ==
整体色温:[K值范围]
冷暖倾向:[暖/中性偏暖/中性/中性偏冷](全片不切换)
饱和度基调:[低/中低]
允许点缀色:[具体色 + 面积上限](方案 B 时填写,方案 A 时填"无")
禁止点缀色:[防打架色]

== 光线参数 ==
光源类型:[自然窗光/漫射自然光]
色温:[K值范围]
光质:**强单侧方向光**(场景图必须;详见 G1 §5.2 场景图光线铁律)
方向:有明确单侧主光源
阴影/高光:**必须有明确投影**(场景图必须)
禁用光:[列出]

== 材质白名单 ==
[列出本风格允许的材质类别]

== 材质黑名单 ==
[列出禁止材质]

== 道具方向 ==
[只定方向,不列具体清单]

== 视觉黑名单 ==
[文字/水印/重复产品/穿帮等通用项 + 风格专属禁项]

== 摄影触发词 ==
[标准英文触发词组,9张图统一使用]
```

## 四、关键决策原则

### 4.1 全片核心落点(重要)
- 所有有落点的图(除尺寸图和特写)必须使用同一种核心落点
- 这是 9 张图成套的核心保证
- 落点选择标准:**存在感低,质感高**
- 避免:粗犷木纹 / 反光石材 / 深色 / 鲜明纹理 / 高饱和
- 优先:细腻匀质木纹 / 哑光浅色陶板 / 素色亚麻 / 浅色无纹石板 / 白大理石

### 4.2 色温与材质的避坑关系

不同材质对色温敏感度不同,定色温时必须考虑产品材质:

| 产品材质 | 色温建议 | 避坑要点 |
|---|---|---|
| 暖色系(陶土/木质/暖金) | 5000K–5300K | 防止暖黄叠加变浊 |
| 冷色系(玻璃/冷金属) | 5300K–5500K | 防止冷调过度 |
| 镜面金属 | 5300K–5500K + 严控暖黄 | 暖黄会让金属变廉价 |
| 纯黑哑光 | 5300K–5500K + 防过曝 | 过曝会让黑色变灰 |
| 纯白瓷 | 5200K–5400K | 防止白色染色偏 |
| 通透玻璃 | 5300K–5500K | 防止环境色污染 |
| 做旧铜/做旧金 | 5300K–5500K | 防止暖黄让旧色变脏 |
| 黑白金组合 | 5300K–5500K + 严控暖黄 | 同时保护三种色相 |

负面词必须根据产品材质追加针对性禁令。例如镜面金/做旧金/含金元素的产品,负面词必须包含 `no warm yellow cast / no amber tone / no gold light pollution`。

### 4.3 道具方向(只定方向,不列清单)

错误:"客厅茶几必放一束蒲苇 + 一本书 + 一个陶瓷碗"
正确:"低饱和天然材质,符合海岸艺术陈设的真实生活逻辑,不抢产品视觉中心"

## 五、风格包自检

| 自检项 | 通过条件 |
|---|---|
| 完备性 | 所有必含字段已填,无空项 |
| 全片核心落点 | 已定义且符合"存在感低质感高"原则 |
| 具体性 | 每项 Nano Banana 可直接执行 |
| 自洽性-色温 | 色温与光线一致 |
| 自洽性-色温与产品材质 | 已考虑产品材质的色温避坑 |
| 自洽性-材质 | 黑白名单无冲突 |
| 自洽性-色彩 | 点缀色与主色盘不打架 |
| **60-30-10 方案锁定** | **已正确判断产品色彩强度,锁定方案 A 或方案 B** |
| **60-30-10 强色防呼应** | **方案 A 时,允许点缀色字段填"无",未塞任何呼应物** |
| **60-30-10 弱色载体多样** | **方案 B 时,9 张图呼应物载体不重复(防"九本书"症)** |
| 光线方向性 | 光线参数明确声明"强单侧方向光"(场景图必须) |
| 与锁定风格的逻辑一致性 | 符合风格名内涵 |

任意一项不通过 → 重新生成。

---

## 六、禁止事项

- 禁止套用预设模板
- 禁止字段缺失就开始后续步骤
- 禁止道具方向写成具体清单
- 禁止风格包未通过自检就传递给逐图落地
- 禁止 9 张图生成时绕过风格包另定参数
- 禁止"全片核心落点"字段缺失

# M2-image-production.md 阶段2:全局锁定 + 逐图落地

> 把阶段1的决策包落地为9张可执行的 Image_1 ~ Image_9 英文描述。

---

## 前置条件(强制)

- 阶段1必须已完整输出决策包
- 决策包必须包含产品定位(含叙事面)+ 3候选风格 + 3候选心智方案 + 买家认知摘要

---

## 执行顺序(强制)

```
Step 1:显式继承阶段1决策包
Step 2:锁定最终风格(从3候选中选1)
Step 3:动态生成风格包(调用 D2)
Step 4:风格包自检
Step 5:锁定最终心智组合方案(从3候选中选1)
Step 6:9图职责分配(调用 D1,考虑叙事面)
Step 7:职责分配预校验
Step 8:逐图生成 Image_1 ~ Image_9(调用 G1)
Step 9:逐图三道审核(第一铁律 / 视觉哲学 / 风格包对账)
Step 10:总校验
Step 11:输出最终结果
```

---

## Step 2:锁定最终风格

锁定优先级:
1. 产品最强信号(I1 引擎已识别)
2. 与买家认知摘要的"风格认知方向"对齐
3. 必消顾虑的对冲能力

铁律:禁止在3候选之外另选;锁定后全片不得漂移。

---

## Step 3:动态生成风格包

调用 `D2-style-package.md`。风格包是 9 张图视觉一致性的唯一保证。

---

## Step 4:风格包自检

| 自检项 | 要求 |
|---|---|
| 完备性 | 覆盖所有必要字段 |
| 具体性 | 每项可被 Nano Banana 直接执行 |
| 自洽性 | 色温/材质/色彩互不冲突 |
| 与锁定风格的逻辑一致性 | 符合风格名内涵 |
| 落点统一原则 | 已定义"全片核心落点" |

任意一项不通过 → 重新生成。

---

## Step 5:锁定最终心智组合方案

从阶段1的3个候选方案中选1个锁定。

锁定优先级:
1. 用户输入的"转化目标"
2. 买家认知摘要的核心使用场景
3. 产品类型的展示逻辑

铁律:禁止在3候选之外另选;锁定后职责清单不得增删。
锁定的是"职责权重分布"和"Image_1强位职责",其余7张职责无顺序意义。

---

## Step 6:9图职责分配

调用 `D1-duty-pool.md`。

### 分配铁律
- Image_2 = 尺寸图(硬锁,不可移位)
- 9张图职责互不重复(同一种展示只做一张)
- 至少1张人物交互图
- 默认场景+置景混搭(场景3-4张/置景4-5张)
- 落点统一(置景类共用同一种,场景类按空间自然变化)
- **如叙事面=仅正面,所有图必须正面或正面±15°,禁止后角/纯侧面**
- 场景空间不得全部相同
- 景别不得全部相同
- 具体空间从阶段1"自然归属空间"取

### 分配表格式

| 编号 | 职责 | 类型(场景/置景)| 落点/空间 | 道具/状态 | 景别+机位 | 人物 |
|---|---|---|---|---|---|---|
| Image_1 | | | | | | |
| Image_2 | 尺寸图(硬锁) | 置景 | 标准底 | 参照物 | 正侧面平视 | 无 |
| ... | | | | | | |

---

## Step 7:职责分配预校验

| 校验项 | 通过条件 |
|---|---|
| Image_2 硬锁 | 是尺寸图 |
| 职责去重 | 9张职责无重复 |
| 交互图数量 | ≥1张 |
| 场景多样性 | ≥3个不同空间 |
| 景别混搭 | ≥3种不同景别 |
| 落点统一 | 置景类用同一落点 |
| 叙事面合规 | 仅正面型禁后角 |
| 空间合规 | 全部空间在归属空间内 |

任意一项不通过 → 重新分配。

---

## Step 8:逐图生成

调用 `G1-image-writing.md` 的写法规范。

每张图只输出一个字段:`Image_N` = "英文描述句"。

---

## Step 9:逐图三道审核(每张图生成后立即执行)

### 9.1 第一铁律审核(最高优先级)
- 产品本体描述是否极简?
- 是否有 `the exact same [类别] from the reference image` 前缀?
- 负面词是否含 `no color change to the product`?
- 是否无颜色/纹理/造型/工艺/尺寸词?
- 细节图是否只描述镜头意图而非产品外观?

### 9.2 视觉哲学审核(详见 G1)
- 道具是否按真实生活大小自然存在?
- 主配关系是否靠焦点/构图/光线/站位建立?
- 描述是否像真实摄影师所写?

### 9.3 风格包对账
- 主色盘是否在边界内?
- 色温/光线/材质是否符合风格包?
- 落点是否符合"全片核心落点"?
- 摄影触发词是否一致?

任意一项不通过 → 该图重写。

---

## Step 10:总校验(9张全部完成后)

| 校验项 | 通过条件 |
|---|---|
| 第一铁律全片覆盖 | 9张全部通过 |
| 视觉哲学全片覆盖 | 9张全部通过 |
| Image_2 尺寸图 | 含 in/cm + 参照物 |
| 职责去重 | 互不重复 |
| 场景多样性 | ≥3个不同空间 |
| 景别混搭 | ≥3种不同景别 |
| 人物交互 | ≥1张 |
| 色系/光线/触发词一致性 | 9张统一 |
| 落点统一 | 置景类共用 |
| 叙事面合规 | 仅正面型无后角 |

任意一项不通过 → 定位到对应图重写。

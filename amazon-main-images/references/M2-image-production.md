# M2-image-production.md 阶段2:全局锁定 + 逐图落地

> 把阶段1的决策包落地为9张可执行的 Image_1 ~ Image_9 英文描述。

---

## 前置条件(强制)

- 阶段1必须已完整输出决策包
- 决策包必须包含产品定位(含叙事面)+ 3候选风格 + 3候选心智方案 + 买家认知摘要

---

## 执行顺序(强制)

```
Step 1:显式继承阶段1决策包
Step 2:锁定最终风格(从3候选中选1)
Step 3:动态生成风格包(调用 D2)
Step 4:风格包自检
Step 5:锁定最终心智组合方案(从3候选中选1)
Step 6:9图职责分配(调用 D1,考虑叙事面)
Step 7:职责分配预校验
Step 8:逐图生成 Image_1 ~ Image_9(调用 G1)
Step 9:逐图三道审核(第一铁律 / 视觉哲学 / 风格包对账)
Step 10:总校验
Step 11:输出最终结果
```

---

## Step 2:锁定最终风格

锁定优先级:
1. 产品最强信号(I1 引擎已识别)
2. 与买家认知摘要的"风格认知方向"对齐
3. 必消顾虑的对冲能力

铁律:禁止在3候选之外另选;锁定后全片不得漂移。

---

## Step 3:动态生成风格包

调用 `D2-style-package.md`。风格包是 9 张图视觉一致性的唯一保证。

---

## Step 4:风格包自检

| 自检项 | 要求 |
|---|---|
| 完备性 | 覆盖所有必要字段 |
| 具体性 | 每项可被 Nano Banana 直接执行 |
| 自洽性 | 色温/材质/色彩互不冲突 |
| 与锁定风格的逻辑一致性 | 符合风格名内涵 |
| 落点统一原则 | 已定义"全片核心落点" |

任意一项不通过 → 重新生成。

---

## Step 5:锁定最终心智组合方案

从阶段1的3个候选方案中选1个锁定。

锁定优先级:
1. 用户输入的"转化目标"
2. 买家认知摘要的核心使用场景
3. 产品类型的展示逻辑

铁律:禁止在3候选之外另选;锁定后职责清单不得增删。
锁定的是"职责权重分布"和"Image_1强位职责",其余7张职责无顺序意义。

---

## Step 6:9图职责分配

调用 `D1-duty-pool.md`。

### 分配铁律
- Image_2 = 尺寸图(硬锁,不可移位)
- 9张图职责互不重复(同一种展示只做一张)
- 至少1张人物交互图
- 默认场景+置景混搭(场景3-4张/置景4-5张)
- 落点统一(置景类共用同一种,场景类按空间自然变化)
- **如叙事面=仅正面,所有图必须正面或正面±15°,禁止后角/纯侧面**
- 场景空间不得全部相同
- 景别不得全部相同
- **机位强制分化**:9 张图的"机位高度 + 观察方位 + 镜头距离"三个变量必须形成 ≥3 种组合,禁止全片 `eye level front-center`(详见下方机位分化变量表)
- 具体空间从阶段1"自然归属空间"取
- 规格类职责(尺寸图)按 S1 前置条件判断
- **如已激活 B1(A 类)**:每张图必须标注装/空状态,Step 6 完成后强制执行 B1 §五 装/空分配总表
- **如已激活 B2(B 类)**:每张图必须标注空瓶/满花/半满状态,Step 6 完成后强制执行 B2 §五 花材分配总表

### 机位分化变量表(强制覆盖)

9 张图必须从下表中分别取值,同一组合(高度+方位+距离)全片最多出现 2 次:

| 维度 | 可选值 |
|---|---|
| **机位高度** | 30° 高俯 / 15-20° 微俯 / 平视 / 5° 微仰 |
| **观察方位** | 正前 / 三分之四左 / 三分之四右 / 纯侧面(左) / 纯侧面(右) |
| **镜头距离** | 较近 / 中距离 / 远距离 / 远距离压缩 |

**写法要求**:每张图的 prompt 必须显式包含
1. `camera angled [具体度数] [up/down]` 或 `camera at eye level`
2. `shown from a [方位] angle so the camera sees [描述]`
3. `shot from a [距离描述] with a medium telephoto lens`

禁止用模糊词如 `slightly above eye level` 替代具体度数。

### 分配表格式

| 编号 | 职责 | 类型(场景/置景)| 落点/空间 | 道具/状态 | 景别+机位 | 人物 |
|---|---|---|---|---|---|---|
| Image_1 | | | | | | |
| Image_2 | 尺寸图(硬锁) | 置景 | 标准底 | 参照物 | 正侧面平视 | 无 |
| ... | | | | | | |

## Step 9:逐图三道审核(每张图生成后立即执行)

### 9.1 第一铁律审核(最高优先级)
- 产品本体描述是否极简?
- 是否有 `the exact same [类别] from the reference image` 前缀?
- 负面词是否含 `no color change to the product`?
- **整个 prompt(全段扫描,不只是产品锚定句)是否无颜色/纹理/造型/工艺词描述产品本体?**
- **光线段落 B 组(光穿透)是否点出了产品颜色或材质?**
  (`mint green portion / glass body / translucent gemstone / amber core` 等复合违规是高频陷阱)
- **道具段落是否误把产品描述词用到了产品上?**
  (如 `a clear glass decanter beside the glass vase` 中后一个 glass 违规)
- 细节图是否只描述镜头意图而非产品外观?
- **详细判断原则与违规模式见 G1 §11.1,审核时必须按其流程逐项扫描**

### 9.2 视觉哲学审核(详见 G1)
- 道具是否按真实生活大小自然存在?
- 主配关系是否靠焦点/构图/光线/站位建立?
- 描述是否像真实摄影师所写?

### 9.3 风格包对账
- 主色盘是否在边界内?
- 色温/光线/材质是否符合风格包?
- 落点是否符合"全片核心落点"?
- 摄影触发词是否一致?

### 9.4 镜头语言硬审核(执行 SKILL H13/H14/H15)

**这一节独立于 G1 §十二 后置审核,是 M2 层的强制关卡,不可豁免。**

对每张非规格图(Image_2 除外)逐项扫描:

| 审核项 | 硬约束 | 通过条件 |
|---|---|---|
| H13 焦段光圈 | SKILL §二 H13 | prompt 含逐字 `85mm` 或类型2允许的 `50mm` + 逐字 `f/2.0`。**出现 f/2.5 / f/2.8 / f/4 / 35mm / 70mm / 100mm / 135mm / macro 即不通过** |
| H14 焦点锁定 | SKILL §二 H14 | prompt 含 `focus locked on the [产品类别]` 结构 |
| H14 前后景分离 | SKILL §二 H14 | prompt 含 `strong foreground-background separation` 逐字 |
| H15 背景 bokeh | SKILL §二 H15 | 背景句只写色调+形状+光线倾向,**不含任何具体道具名 + 材质 + 位置组合**。判断方法见 G1 §6.2 "核心判断标准" |

任意一项不通过 → **该图立即重写**,不给"其他部分都写好了就这一点小问题"的豁免。

### 9.5 规格图豁免说明
Image_2 尺寸图**豁免** 9.1 中的部分项、9.2 视觉哲学、9.4 镜头语言审核,但必须通过 S1 §5 规格图专属审核。

任意一项不通过 → 该图重写。

## Step 10:总校验(9张全部完成后)

| 校验项 | 通过条件 |
|---|---|
| 第一铁律全片覆盖 | 9张全部通过 |
| 视觉哲学全片覆盖 | 9张全部通过 |
| **H13 镜头焦段光圈全片统一** | **9 张非规格图全部 85mm(类型2 允许 50mm)+ f/2.0,无一张偏离** |
| **H14 焦点锁定+前后景分离全片覆盖** | **9 张非规格图全部含对应逐字声明** |
| **H15 背景 bokeh 句式全片合规** | **9 张非规格图背景句无一张含具体道具名+材质+位置组合** |
| Image_2 尺寸图 | 含 in/cm + 参照物,无镜头语言字段污染 |
| 职责去重 | 互不重复 |
| 场景多样性 | ≥3个不同空间 |
| 景别混搭 | ≥3种不同景别 |
| 人物交互 | ≥1张 |
| 色系/光线/触发词一致性 | 9张统一 |
| 落点统一 | 置景类共用 |
| 叙事面合规 | 仅正面型无后角 |

任意一项不通过 → 定位到对应图重写。

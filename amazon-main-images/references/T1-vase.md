## 一、适用范围

- 仅适用于花瓶类产品(B类)
- 由 M1 Step 2 识别产品类型=B 时自动激活
- 不适用于 A / C / D 类

---

## 二、核心原则

1. 花瓶存在两种视觉状态:**空瓶**(展示工艺) / **满花**(展示功能)。
2. 空瓶 vs 满花的分配不是临时决定,必须由 I2 心智方案在阶段1就锁定。
3. 花材是花瓶类的功能表达,不是普通道具,不计入风格包"道具方向"约束。
4. 同一花材全片最多出现 2 次,第 3 次强制更换。

---

## 三、花瓶呈现模式(I2 推导补充)

> 主图无叙事,所以不存在 R05 的"叙事锚点"。
> 主图的等价物是:I2 心智方案必须显式声明本片的"花瓶呈现模式"。

### 三种呈现模式

| 模式 | 空瓶职责数 | 满花职责数 | 适用心智 |
|---|---|---|---|
| **纯工艺型** | 多数(5–7张)| 少数或 0 | 工艺信任 / 艺术品质感 |
| **纯生活型** | 少数(1–2张) | 多数(5–7张) | 居家代入 / 礼品转化 |
| **均衡型** | 3–4张 | 3–4张 | 综合心智 |

### I2 推导铁律(B类追加)

- I2 输出 3 个候选方案时,3 个方案的呈现模式不得全部相同
- 每个方案必须显式标注:`花瓶呈现模式:[纯工艺型/纯生活型/均衡型]`
- 全片至少 1 张空瓶图(防止全片只见花不见瓶)
- 全片至少 1 张满花图(防止全片证明不了实用性),除非心智明确为"纯艺术品收藏"

## 四、花瓶专属职责(D1 交互组扩展)

> R05 的"商业动作"在视频里是连续的镜头链,在主图里改成**单张图捕捉的动作瞬间**。
> 这些职责追加到 D1 交互组,B 类专用,不影响其他品类。

| 职责名称 | 单图呈现 | 手部 | 通常对应模式 |
|---|---|---|---|
| **双手捧瓶** | 双手从两侧捧住瓶身,展示体量与质感 | 双手 | 任意 |
| **转瓶展示** | 单手扶瓶身侧面,呈现"转动中"的瞬间 | 单手 | 纯工艺型 |
| **瓶口俯览** | 手轻扶瓶身,俯角拍摄展示瓶口造型 | 单手 | 纯工艺型 |
| **插花瞬间** | 单手持一支花茎,正要送入瓶口的瞬间 | 单手 | 纯生活型/均衡型 |
| **整束入瓶** | 双手捧一束花,正要放入宽口的瞬间 | 双手 | 纯生活型 |
| **注水瞬间** | 单手持壶/杯,水线正在落入瓶口的瞬间 | 单手 | 纯生活型/均衡型 |
| **落位归置** | 双手将瓶放至台面,刚刚松手的瞬间 | 双手 | 任意 |
| **礼品递送** | 双手捧瓶向镜头递出 | 双手 | 礼品转化心智 |

### 职责选用规则

- 全片至少 1 张交互职责(继承 D1 总规则)
- 同一种交互职责全片只用 1 次
- "注水瞬间"和"插花瞬间"不可同时出现(避免动作语义冲突)
- 交互职责的动作必须与该图的空瓶/满花状态自洽:
  - 注水瞬间 → 该图必为空瓶或半空(瓶里不能已经有花)
  - 插花瞬间 → 该图为半满状态(瓶里已有部分花,正在加新的)
  - 整束入瓶 → 该图为空瓶状态(瓶里还没有花)
  - 落位归置 → 空瓶或满花皆可
  - 双手捧瓶/转瓶/瓶口俯览 → 通常空瓶
  - 礼品递送 → 通常空瓶或精致小束

### 场景→花材推导原则(动态推导,禁止套用清单)

- **光线** → 自然窗光强 → 鲜花;漫射柔光 → 干花

- **职责类型** → 工艺类 → 无花或极少;场景类/交互类 → 体量饱满
- **风格包色盘** → 花材色必须在风格包允许点缀色范围内,严禁高饱和打架

## 六、空瓶 vs 满花的构图策略

### 空瓶状态

- 构图焦点:瓶身轮廓、纹理、釉色、瓶口造型
- 瓶口必须完整可见,不被遮挡
- 光线优先打透瓶体或斜打表面,展示材质质感
- 适用职责:工艺组全部 / 借景型置景 / 尺寸图 / 部分场景图

### 满花状态

- 构图焦点:花材+瓶体的整体组合
- 允许花头轻微遮挡瓶口边缘(饱满的自然结果)
- 但瓶身中下段必须清晰可见,不能被花材完全遮挡
- 光线兼顾花材色彩与瓶体质感
- 适用职责:场景类多数 / 交互类(插花/整束/注水)

### 半满状态

- 仅用于"插花瞬间"职责
- 瓶里已有少量花,手部正在加新的
- 构图焦点同时落在瓶口、已有花材、手部动作三者

## 七、花材体量基准(按瓶口宽度)

> 主图体系下,花材描述聚焦视觉饱满度,不写支数。
> 但内部判断花量要参考瓶口宽度。

| 瓶口宽度 | 内部判断的花量参考 | 目标视觉 |
|---|---|---|
| 窄口(≤5cm) | 3–5 主花 + 1–2 配花 | 花茎紧密聚拢,花头自然散开高于瓶口 |
| 中口(5–10cm) | 5–8 主花 + 2–3 配花 | 花束饱满充盈,花头覆盖瓶口大部分 |
| 宽口(≥10cm) | 8–12 主花 + 3–5 配花 | 花束丰盛溢出,花头几乎完全覆盖瓶口 |

### 视觉铁律(仅对满花/半满图生效)

> 空瓶图不受本节约束,空瓶图执行第六节"空瓶状态构图策略"。

- 满花图禁止"瓶口大面积空置只插 1–2 支孤花"的画面
- 花头总面积 ≥ 瓶口面积 70%
- 花束高度 ≥ 瓶身高度 0.6 倍
- 花头允许轻微遮挡瓶口边缘(饱满自然结果)
- 特写/微距景别可只拍局部花材,不受此限

## 八、瓶口遮挡关系(按景别)

| 景别 | 瓶口可见度要求 |
|---|---|
| 全景 / 中景 | 花材可覆盖瓶口,瓶身轮廓清晰可辨 |
| 近景 | 瓶口边缘至少露出 30%,让买家看到瓶口造型 |
| 特写 / 微距 | 拍瓶口 → 无花;拍花材 → 瓶口作为背景部分可见 |
| 交互手部图 | 瓶口必须清晰可见,买家需看到手部动作如何作用于瓶口 |

---

## 九、空瓶图的负面词追加

> 空瓶图必须在 G1 第八节通用负面词之外追加:

```
no flowers, no plants, no stems, no liquid inside, no filling
```

> 满花图无需追加负面词,体量由第六节构图策略 + 第七节视觉铁律正面驱动。

## 十一、全局禁止(T1 专属)

- 禁止 Image_2 尺寸图带花
- 禁止全片只有满花图(看不到瓶)
- 禁止全片只有空瓶图(除非纯工艺型心智)
- 禁止同一花材全片出现 ≥ 3 次
- 禁止花材色与风格包点缀色冲突

- 禁止交互动作与瓶内状态自相矛盾(如注水时瓶里已经满花)

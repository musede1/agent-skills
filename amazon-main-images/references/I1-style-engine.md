# I1-style-engine.md 风格候选推导引擎(主图版)

> 为家居摆件产品自动选出 3 个候选风格,Top1 尽量是真正最优解。
> 所有风格候选均为动态推导,禁止套用预设风格库。

---

## 一、第一铁律(主图专属)

```
主图风格必须在"明亮安全区"内推导。
暗调 / 戏剧化 / 高对比 / 暗黑收藏 风格默认排除。

仅当产品题材本身明确指向暗黑时,才允许暗调候选,例如:
  · 哥特装饰品 / 万圣节摆件 / 暗黑神秘收藏品

明亮安全区不只有奶油橡木一种,常见组合包括:
  · 暖奶油+漂白橡木+亚麻(温暖乡村方向)
  · 白大理石+金色配件+米白(现代轻奢方向)
  · 浅灰白+浅胡桃木+米白(现代极简方向)
  · 纯白+浅木+干花(温柔治愈方向)
  · 暖米白+陶土+棕木(自然侘寂方向)
  · 浅灰+水泥+黑铁(工业现代方向)
  · 等等

明亮安全区的核心定义:背景明亮、整体低饱和、产品是画面唯一高饱和点。
不限定具体材质组合,根据产品最强信号自由推导。
```

---

## 二、最强信号识别(核心方法)

风格判断不走固定优先级,而是先识别"最强信号",再以最强信号为锚点推导风格。

### 判断顺序

```
Step 1:扫描产品所有维度信号
  · 题材(海洋/动物/人物/花鸟/节庆/抽象等)
  · 材质组合(单一材质 vs 强材质组合)
  · 工艺(手工吹制/手雕/手绘等)
  · 气质(温柔/现代/复古/萌趣等)
  · 色彩与造型

Step 2:识别"无可替代的最强信号"
  判断标准:如果换掉这个信号,产品就完全不是这个产品了

Step 3:以最强信号为锚点推导风格
  其他维度作为修饰和细节

Step 4:输出 3 个候选,Top1 锚定最强信号,Top2/Top3 提供差异化
```

### 最强信号优先级(仅作冲突仲裁,不作默认排序)

| 优先级 | 信号类型 | 例子 |
|---|---|---|
| 1 | 文化/节庆/宗教符号 | 万圣节南瓜 / 圣诞袜 / 十字架 / 招财猫 |
| 2 | 强材质组合 | 镜面金气球狗 / 哑光黑白金情侣 / 手工吹制玻璃 |
| 3 | 强题材 | 海螺(海洋)/ 大嘴鸟(动物+实用)|
| 4 | 工艺特征 | 手工吹制 / 手雕木 |
| 5 | 气质综合 | 普通陶瓷瓶 / 普通木摆件 |

## 三、产品信号读取

| 信号维度 | 提取内容 |
|---|---|
| 产品类型 | 摆件 / 花瓶 / 雕像 / 容器 / 墙饰 |
| 题材属性 | 海洋 / 动物 / 人物 / 花鸟 / 节庆 / 抽象 |
| 材质组合 | 单一材质 / 强材质组合(双色/三色/混合)|
| 强符号识别 | 海螺/贝壳/树/天使/南瓜/十字/动物形象 |
| 气质属性 | 艺术 / 温柔 / 治愈 / 神秘 / 萌趣 / 复古 / 轻奢 |
| 使用定位 | 日常陈设 / 礼品向 / 收藏向 / 节庆向 |
| 表面处理 | 哑光 / 高光 / 透明 / 做旧 / 镜面 |
| 色彩与造型 | 主色冷暖、饱和度、几何/有机/Q版/雕塑感 |
| 叙事面 | 全方位 / 仅正面(影响候选风格的拍法约束)|

## 四、强优先规则

### 4.1 文化/节庆符号必胜
当产品有明确节庆/宗教/文化符号时,题材方向永远优先,即便材质很强也不动摇。

### 4.2 强材质组合优先
当产品材质组合本身就是定位(镜面金/侘寂陶/手工玻璃),材质方向优先。

### 4.3 数据驱动加权
若《买家认知摘要》提供了"风格认知方向",该方向的候选风格匹配度自动+1星。

### 4.4 差评对冲
若有显著差评,候选风格中至少 1 个必须能直接对冲该差评。

### 4.5 常见误判禁止
- 禁止只因"金色/电镀"就判轻奢
- 禁止只因"白色/哑光"就判极简
- 禁止只因"深色"就判暗调收藏
- 禁止把所有树脂归萌趣
- 禁止把所有玻璃归现代

## 五、候选生成

### 5.1 命名规范
- 风格名称必须具体可理解
- **禁止包含具体空间名词**(如"玄关风/客厅风/卧室风"),应只描述气质和材质方向
- 这样才能兼容场景图和置景图两种表达

### 5.2 命名公式参考
```
候选风格 = 题材主轴 + 气质主轴 + 材质方向 + "陈设风"
```

示例:
- 海岸艺术陈设风
- 现代轻奢陈设风
- 温暖乡村陈设风
- 现代极简画廊陈设风
- 自然侘寂陈设风

---

## 六、评分与排序

### 6.1 评分维度

| 维度 | 权重 |
|---|---|
| 与最强信号的对齐度 | 最高 |
| 与买家认知摘要的对齐度 | 很高 |
| 必消顾虑的对冲能力 | 高 |
| 使用定位与风格落点的匹配度 | 高 |

### 6.2 输出角色分工

- 候选1:最符合最强信号 + 最匹配买家认知
- 候选2:最稳妥商业化(送礼/居家代入感最强)
- 候选3:有差异但合理

铁律:
- 3个候选不得是同一风格的近义词
- 3个候选必须全部在明亮安全区内
- 3个候选名称都不得包含空间词

---

## 七、输出格式

```
【候选风格】
| 编号 | 风格名称 | 匹配理由 | 匹配度 |
|---|---|---|---|
| 1 | [风格名] | [基于最强信号+买家认知的具体理由] | ⭐⭐⭐⭐⭐ |
| 2 | [风格名] | [理由] | ⭐⭐⭐⭐ |
| 3 | [风格名] | [理由] | ⭐⭐⭐ |
```

匹配理由禁止只写空话。

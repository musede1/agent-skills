# R05-vase-flower-guide.md 花瓶类（B类）专项规则

> 本文件是花瓶类产品的完整专项规则文件。
> 涵盖：叙事规则 / 专属商业动作 / 花材规则 / 构图专项。
> 未在本文件覆盖的规则，全部继承现有体系（R01–R04 / R06–R07 / W01–W03 / P1–P3）。

---

## 一、适用范围

仅适用于花瓶类产品（B类）。
不适用于普通容器类、储物罐、收纳罐（A类）。
不适用于纯装饰摆件（C类）或墙饰（D类）。

---

## 二、核心原则

1. 花材是花瓶类的功能表达，不是普通 Decor。
2. 花材搭配必须服务花瓶体量、开口、颜色与空间气质。
3. 不启用同类陈设组默认逻辑。
4. 同一花材（单支或组合）全片最多出现 2 次，第 3 次必须强制更换。

---

## 三、叙事锚点池（替代SKILL.md硬锁）

> 本节由 E2-narrative-engine.md 在推导B类叙事主线时调用。
> E2从锚点池中选取或组合，动态推导3条差异化叙事线。
> 禁止每片必选同一个锚点，3条候选线的核心锚点必须各不相同。

### 可用叙事锚点

| 锚点名称 | 叙事围绕什么展开 | 典型镜头特征 |
|---|---|---|
| 插花动作 | 从空瓶到插满花的使用过程 | 有手部动作，花材逐步入瓶 |
| 光影穿透 | 光线穿过玻璃材质的色彩与折射变幻 | 空瓶为主，强调光影事件 |
| 空间漫游 | 同一花瓶在不同房间的陈设效果 | 多场景切换，花瓶状态可变 |
| 工艺质感 | 手工吹制/窑变/纹理的独一无二感 | 多角度特写，转瓶展示 |
| 花材搭配 | 不同花材与花瓶的搭配效果展示 | 多组花材切换，瓶体不变 |
| 使用流程 | 完整使用链（注水→插花→整理→归位） | 有因果链，类似A类逻辑 |
| 礼品仪式 | 开箱→欣赏→摆放的送礼场景 | 有包装/礼盒元素 |

### 执行规则

- E2推导时从池中选取1–2个锚点作为每条叙事线的核心
- 3条候选线的核心锚点必须各不相同
- 允许同一条线内组合多个锚点（如"光影穿透+工艺质感"）
- 插花动作不再是必选项，但始终在候选池中
- 未列入池中的方向，E2可根据产品信号动态新增

## 四、花瓶专属商业动作（追加至R03动作库）

> 以下动作仅适用于B类花瓶，追加使用，不影响A/C/D类现有动作。
> B类同时可使用R03通用动作中的"轻触调整"、"静态材质展示"、"场景收束"。

### 动作清单

| 动作名称 | 说明 | 手部 |
|---|---|---|
| **插花入瓶** | 手持花材从上方逐支插入瓶口 | 单手 |
| **整束入瓶** | 手握一整束花直接放入瓶口，展示宽口实用性 | 单手或双手 |
| **注水入瓶** | 从壶/杯缓缓倒水入瓶，水线上升，展示玻璃透明质感 | 单手持壶 |
| **转瓶展示** | 手轻触瓶身缓缓转动，展示多面工艺/纹理 | 单手 |
| **花材整理** | 手轻轻调整已插好的花材角度和高度 | 单手 |
| **瓶口展示** | 俯拍角度，手倾斜瓶身展示瓶口造型（荷叶边/窑变边） | 单手或双手 |
| **双手捧瓶** | 双手从台面捧起花瓶，传递体量感与手工质感 | 双手 |
| **落位归置** | 双手捧瓶从画面外入镜，放到台面指定位置 | 双手 |

### 使用规则

- 每个动作可**单独使用**（独立出现在某个镜头）
- 也可**组合使用**（串成因果链，在同一镜头内或相邻镜头间完成）
- 全片不得重复同一个商业动作（继承R03现有规则）
- 由E2根据叙事线自行决定用单个还是组合，不强制

### 动作组合参考（非强制，E2可自由组合）

| 组合类型 | 动作链 | 适用叙事方向 |
|---|---|---|
| 完整使用流程 | 落位归置 → 注水入瓶 → 插花入瓶 → 花材整理 | 使用流程型叙事线 |
| 轻量插花 | 注水入瓶 → 整束入瓶 | 单镜头内快速完成 |
| 纯欣赏展示 | 双手捧瓶 → 转瓶展示 | 工艺质感型叙事线 |
| 落位收官 | 落位归置 → 花材整理 | 收官镜头 |
| 瓶口特写 | 瓶口展示 → 注水入瓶 | 展示瓶口造型+功能 |

> 以上为参考方向，E2可根据产品信号自由组合或创建新链，不限于此。

## 五、花材全片分配总表（B类强制前置执行锁）

> 必须在叙事主线锁定之后、第一个Shot开始之前完整填写并通过自检。
> 未通过自检禁止开始写任何Shot内容。

### 填写顺序（强制）

```
Step 1：确定每个Shot的场景
Step 2：根据场景推导适配花材
Step 3：填写总表
Step 4：花材去重自检
Step 5：全部通过 → 开始写Shot内容
```

### 场景→花材推导原则

花材内容由AI根据以下四个维度动态推导，禁止套用固定清单：

**① 光线强度** → 自然光充足的场景（窗台/阳台）适合有生命感的鲜花；光线柔和的室内场景（茶几/书桌/餐边柜）适合耐久的干花

**② 空间使用频率** → 长期陈列场景选干花；仪式感/节庆场景选鲜花

**③ 场景视觉复杂度** → 场景道具已较丰富时，花材宜少且克制；场景干净简洁时，花材可适当有体量感

**④ 镜头职责** → 钩子/材质特写镜头，花材极少或无，让釉色和造型主导；情绪高点/场景代入镜头，花材可承担氛围烘托

> 铁律：先锁定场景 → 再根据以上四个维度推导花材 → 再做去重检查。禁止先定花材再配场景。

### 总表格式

| Shot | 场景 | 花材内容 | 体量状态 | 是否含手部动作 |
|---|---|---|---|---|
| shot_1 | | [花材 / 无花] | [饱满溢出/密集充盈/适度丰盈/—] | 是/否 |
| shot_2 | | [花材 / 无花] | | 是/否 |
| shot_3 | | [花材 / 无花] | | 是/否 |
| shot_4 | | [花材 / 无花] | | 是/否 |
| shot_5 | | [花材 / 无花] | | 是/否 |
| shot_6 | | [花材 / 无花] | | 是/否 |

### 花材去重自检

| 花材名称 | 计划出现Shot | 累计次数 | 超限?(>2) |

## 六、花材输出格式

每个Shot的花材描述必须明确写出：
- 多少支什么花
- 搭配多少支什么配花（如有）
- 花材风格（现代简约 / 自然度假 / 复古文艺 等）
- 体量状态（饱满溢出 / 密集充盈 / 适度丰盈，禁止写"克制留白"或"几支点缀"）

## 七、花材体量控制

### 体量基准原则
花材体量由花瓶开口宽度决定，不由固定支数决定。
目标状态：花束从瓶口溢出，视觉上饱满丰盈，花头高度约为瓶身高度的0.8–1.2倍。

### 体量档位（按花瓶开口宽度判定）

| 开口宽度 | 体量要求 | 目标视觉状态 |
|---|---|---|
| 窄口（≤5cm） | 3–5支主花 + 1–2支配花 | 花茎紧密聚拢，花头自然散开高于瓶口 |
| 中口（5–10cm） | 5–8支主花 + 2–3支配花 | 花束饱满充盈，花头密集覆盖瓶口大部分面积 |
| 宽口（≥10cm） | 8–12支主花 + 3–5支配花 | 花束丰盛溢出，花头几乎完全覆盖瓶口且向外自然展开 |

### 铁律
- 禁止出现"花瓶口大面积空置、只插1–2支孤零零的花"的画面
- 花头总面积必须覆盖瓶口面积的70%以上
- 花束高度不得低于瓶身高度的0.6倍（太矮显得敷衍）
- 允许花头轻微遮挡荷叶边/瓶口边缘，这是饱满的自然结果
- 特写/微距镜头除外，该景别可只拍局部花材

---

## 八、空瓶与满花构图策略

> 花瓶产品存在两种视觉状态，构图策略不同。
> 由叙事锚点决定全片空瓶/满花的分配比例，不写死。

### 空瓶状态（强调瓶体工艺与造型）

- 构图焦点：瓶身轮廓、纹理、釉色、瓶口造型
- 瓶口必须完整可见，不被遮挡
- 光线优先打透瓶体，展示材质通透感或表面质感
- 适用锚点：光影穿透 / 工艺质感

### 满花状态（强调花瓶的使用场景与生活感）

- 构图焦点：花材+瓶体的整体组合
- 允许花头轻微遮挡瓶口边缘（饱满的自然结果）
- 但瓶身中下段必须清晰可见，不能被花材完全遮挡
- 光线兼顾花材色彩与瓶体质感
- 适用锚点：插花动作 / 花材搭配 / 空间漫游

### 全片分配规则

- 不强制空瓶与满花的固定比例
- 叙事线以"光影穿透"或"工艺质感"为核心锚点时 → 空瓶镜头可占多数
- 叙事线以"插花动作"或"花材搭配"为核心锚点时 → 满花镜头可占多数
- 全片至少1个镜头展示瓶体本身（空瓶或仅局部花材），避免全片看不清花瓶长什么样

---

## 九、花材与瓶口遮挡关系

| 景别 | 瓶口可见度要求 |
|---|---|
| 全景 / 中景 | 花材可覆盖瓶口，但瓶身轮廓必须清晰可辨 |
| 近景 | 瓶口边缘至少露出30%，让买家看到瓶口造型 |
| 特写 / 微距 | 如拍瓶口 → 无花或花材移出画面；如拍花材 → 瓶口作为背景可部分可见 |
| 手部动作镜头 | 瓶口必须清晰可见（买家需要看到花材如何进入瓶口） |

## 十、E2叙事推导补充规则（B类专项）

> 本节内容由 E2-narrative-engine.md 在推导B类叙事主线时调用。

### 推导执行顺序

```
Step 1：读取产品信号（材质/透明度/开口大小/工艺特征/色彩）
Step 2：从第三节叙事锚点池中，根据产品信号匹配最适合的锚点方向

Step 5：根据锚点从第四节动作库中选取适配的单个动作或动作组合
Step 6：输出3条候选叙事线
```

### 产品信号→锚点匹配参考

| 产品信号 | 优先匹配锚点 |
|---|---|
| 透明/半透明玻璃 | 光影穿透 |
| 多面工艺/多色纹理 | 工艺质感 / 转瓶展示 |
| 宽口设计 | 插花动作 / 整束入瓶 |
| 窄口/细颈 | 光影穿透 / 空瓶工艺 |
| 高辨识度造型（荷叶边/异形口） | 瓶口展示 / 工艺质感 |
| 礼品向/礼盒包装 | 礼品仪式 |
| 大尺寸/重体量 | 落位归置 / 双手捧瓶 |

> 以上为参考方向，不强制映射。E2根据产品信号综合判断。

### B类叙事铁律
- 3条候选线的核心锚点必须各不相同
- 禁止3条线都以插花动作为核心锚点
- 全片至少1个镜头展示瓶体本身（防止全片只看到花看不到瓶）
- 选择"使用流程"锚点时，动作顺序必须符合真实因果（注水在插花之前，不可倒置）

## 十三、花材体量描述规范（image_prompt专项）

> 本节由阶段3生成image_prompt时调用。
> 图像模型对"3支玫瑰"这类支数描述响应极弱，必须用体量描述词+空间关系词强制约束花材密度。

### 强制体量描述词（image_prompt中至少使用2个）
- densely packed / tightly clustered / abundantly full
- overflowing generously / brimming at the rim
- lush voluminous bouquet / generous abundant arrangement

### 强制空间关系描述（至少使用1个）
- blooms rising well above the rim（花头高于瓶口）
- petals nearly covering the entire mouth of the vase（花头覆盖瓶口）
- bouquet volume roughly equal to the vase body（花束体量与瓶身相当）

### 禁止使用的弱描述词
- a few stems / a single bloom / sparse arrangement
- minimalist bouquet / delicate sprig / one or two flowers
- 任何暗示花量少的词汇

### 写法示例
❌ 错误：holding three pale purple hydrangea blooms and one dusty miller stem
✅ 正确：holding a densely packed abundant bouquet of pale purple hydrangeas and dusty miller, tightly clustered and overflowing generously above the ruffled rim, the lush flower volume roughly matching the vase body in size

### 后置校验（阶段3每条含花image_prompt写完后检查）
- 花材描述中是否包含至少2个体量描述词？
- 是否包含至少1个花头与瓶口的空间关系描述？
- 是否使用了任何弱描述词？（发现则必须替换）

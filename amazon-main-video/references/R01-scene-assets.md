# scene-assets.md（优化版）

## 1. 术语定义

### Base
让场景成立，让人一眼认出空间。

### Decor
补氛围、补层次的点缀道具。

### Functional Items
表达产品用途 + 尺度参照。仅 A 类启用。

### 同类陈设组
表达收纳/分类/同体系陈列逻辑。仅 A 类启用。

---

## 2. 执行优先级
先保 Base → 再保 Functional Items → 再保同类陈设组 → 最后决定 Decor 数量。

---

## 3. Base 数量规则

### 前置判断（强制）
- 户外场景 → 不需要 Base
- 室内非场景图（材质特写 / 交互图）→ 不需要 Base
- 室内场景图 → 按景别决定数量

### 室内场景图 Base 数量
| 景别 | Base 数量 |
|---|---|
| 全景 / 中景 | 最少 2 个 |
| 近景 / 中近景 | 最少 1 个，允许 2 个 |
| 特写 / 微距 | 0–1 个 |

### 说明
- Base 内容由 AI 根据场景自然决定，规则只约束数量
- Base 不计入 Decor 数量

---

## 4. Decor 数量规则

| 景别 | A类摆件 | B类花瓶 | C类纯装饰 |
|---|---|---|---|
| 全景 / 中景 | 2–3 件 | 1–3 件 | 1–3 件 |
| 近景 / 中近景 | 1–2 件 | 1–2 件 | 1–2 件 |
| 特写 / 微距 | 0 件 | 0 件 | 0 件 |

### 说明
- Decor 内容由 AI 根据场景与风格自然决定，规则只约束数量
- Decor 不计入 Functional Items 与同类陈设组数量

---

## 4A. Decor 主题一致性规则（强制）

> 核心原则：产品有强主题属性时，Decor必须与产品主题语境一致。
> 买家看到画面时，所有台面元素应指向同一个生活场景联想，不能出现认知分裂。

### 判定逻辑

```
Step 1：读取产品的题材属性和强符号（来自E1信号读取）
Step 2：判断是否存在强主题属性
  有强主题属性 → Decor执行主题匹配
  无强主题属性 → 由AI根据场景自然决定（继承§4现有规则）
```

### 主题→Decor方向参考（非固定清单，AI动态推导）

| 产品主题属性 | Decor方向 | 禁止方向 |
|---|---|---|
| 海洋（贝壳/海螺/珊瑚/鱼） | 白色陶瓷海星、天然海胆壳、漂白浮木、白色珊瑚枝、磨砂玻璃卵石、天然扇贝壳、干燥海扇、干燥蒲苇穗 | 彩色塑料海洋玩具、麻绳、船锚、船舵、鲜艳贝壳 |
| 节庆-万圣节 | 白色/米色南瓜、黑色蜡烛、干燥树枝、暗色调干花 | 彩色塑料南瓜、卡通幽灵、荧光装饰 |
| 节庆-圣诞 | 松果、干燥松枝、白色/金色蜡烛、亚麻桌布 | 廉价闪粉装饰、彩色塑料球 |
| 花鸟植物 | 干燥花材、苔藓球、天然木片、小型陶瓷花盆 | 塑料仿真花、彩色花盆 |
| 宗教/仪式感 | 素色蜡烛、亚麻布、石质小物、无字旧书 | 彩色宗教纪念品、印字物品 |
| 萌趣/礼品 | 牛皮纸、棉绳、简约礼品盒、干花小束 | 塑料包装、荧光纸 |

> 以上为方向参考，AI根据具体产品信号动态推导，不限于此。
> 不在表中的主题，AI基于同一逻辑自行判断。

### 铁律
- 产品有强主题属性时，禁止使用与主题无关的通用道具（精装书、笔记本、眼镜等）
- 主题匹配Decor仍受§4数量规则约束（全景/中景2-3件等）
- 主题匹配Decor质感必须过关：天然材质 > 陶瓷仿制 > 禁止塑料彩色
- 同一道具全片最多出现2次（继承现有去重规则）

## 5. Functional Items 规则（仅 A 类）

### 容量判定（强制前置）
执行 FI 分配前，必须先显式输出容量判定：
> 【托盘/容器容量判定：根据产品图，该产品承载面/开口尺寸约为 XX cm，合理最大装载量为 X 件小物件，全片 FI 数量上限锁定为 X 件】

禁止跳过此步骤直接分配 FI 数量。

### 全片分布
- 全片 6 镜头中，**3–5 个镜头放** Functional Items，**1–3 个镜头不放**
- 不放的镜头通常是：材质特写、纯造型细节、收官镜头

### 数量规则
- 每镜头视觉单元数量由**产品容量**自然决定，以真实装载状态为准
- 镜头间数量**必须有变化**，不能每镜头都一样
- 尽量多放，但至少有 1 个镜头明显少于最大容量
- 放什么、放几件交给 AI 根据场景自决

### 搭配规则（强制）
- FI 内容必须符合**场景使用逻辑**，不同场景放不同物件
  - 玄关：钥匙串、硬币、零钱卡、卡包等
  - 梳妆台：戒指、耳环、发夹、小首饰等
  - 客厅茶几：遥控器、眼镜等
  - 书桌：耳机、U盘、小文具等
- 同一件 FI 物件全片**最多出现 2 次**，避免单调重复
- FI 物件尺寸必须与产品容器开口匹配，放不进去的物件禁止出现
- 同一镜头内 FI 物件之间必须有**生活逻辑关联**，禁止随意混搭

---

## 6. 同类陈设组规则（仅 A 类）

- 数量：0–1 组，每组 2–4 件
- 仅在"柜内收纳 / 分类整理 / 台面成组陈列"镜头中启用
- 动作镜头默认不开启
- 不计入 Decor 数量

---

## 7. 负面提示词（兜底）
前后双台面，中岛分离，像展厅，像样板间，纯摆拍空台面，物品围绕镜头刻意排列，平均铺开，货架售卖感。

---

## 8. 逐镜强制对账（AI执行锁）

> 核心原则：每写完一个Shot立即填表，确认达标后才能继续下一个Shot。
> 禁止全部写完后再统一校验。
> 本节负责"逐镜实时防错"；P2第十二节负责全部Shot完成后的"三项汇总确认"，两者分工不同，必须同时执行。

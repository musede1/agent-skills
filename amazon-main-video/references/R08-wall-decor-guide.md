# R08-wall-decor-guide.md 墙饰类（D类）专项规则

> 本文件只处理墙饰产品与台面产品的差异点。
> 未在本文件覆盖的规则，全部继承现有体系（R01–R07 / W01–W03 / P1–P3）。

---

## 一、D类判定标准

| 子类 | 判定标准 | 叙事继承 |
|---|---|---|
| **D-装饰型** | 以视觉陈列为主，无明确使用动作，挂墙或台面陈列均可 | 继承C类情绪递进逻辑 |
| **D-功能型** | 有明确使用动作（计时/置物/插花等），以挂墙方式实现功能 | 继承A类使用流程逻辑 |

**D-装饰型示例：** 金属墙艺 / 浮雕挂件 / 挂毯 / 镂空墙饰 / 装饰镜 / 画框 / 树脂浮雕挂件

**D-功能型示例：** 挂钟 / 壁挂花插 / 置物壁架 / 壁挂钥匙架

> D-功能型执行时，额外参考 P1-positioning.md 的 A类规则（FI容量判定 / 使用动作顺序）。

## 二、双落点说明

D类产品允许在拍摄中出现两种落点：

```
落点1：挂墙场景 → 产品挂于墙面，交代真实使用状态
落点2：台面陈列 → 产品平放或靠放于台面，用于细节特写或造型展示
```

**规则：**
- 挂墙场景镜头：必须遵循本文件的挂墙场景构图规则
- 台面陈列镜头：按"室内非场景图"处理，不需要Base，继承R01现有规则
- 全片两种落点均可出现，不强制只选一种
- 叙事主线中，挂墙场景镜头不得少于1个（至少让买家看见挂墙状态）

## 三、尺寸档位（D类专属）

> D类按产品展开后的最长边判定，不按高度。
> 同尺寸产品挂墙后视觉体量远大于台面，档位定义与台面产品不同。

| 档位 | 最长边范围 | 适用产品示例 |
|---|---|---|
| **S档** | 15–35cm | 小型浮雕挂件、小装饰镜、迷你挂钟 |
| **M档** | 35–65cm | 中型金属墙艺、中号装饰镜、标准挂钟 |
| **L档** | 65cm以上 | 大型金属墙艺、大幅挂毯、Gallery wall套组 |

## 四、挂墙场景 Base 规则（改写）

> 替代 R01 中台面产品的 Base 规则，仅适用于D类挂墙场景镜头。

### 核心逻辑
墙面本身即是产品的"画布"，不需要台面家具撑起场景。
Base 的职责变为：**让观者一眼认出这是真实居家空间，而非摄影棚白板。**

### Base 内容来源（按优先级）

```
优先级1：下方家具顶线
  → 沙发背、控台顶面、床头板、书柜顶部边缘
  → 作用：交代空间高度关系，建立产品与生活的比例感

优先级2：墙面结构元素
  → 踢脚线、门框边缘、窗框线、墙面分色线
  → 作用：暗示空间结构，不需要整体家具入画

优先级3：墙面质感本身
  → 乳胶漆肌理、木板条纹、砖墙纹路
  → 作用：当产品体量足够大，墙面质感即可撑起空间感
```

### Base 数量规则

| 景别 | Base 要求 |
|---|---|
| 全景 | 必须有下方家具顶线入画（至少局部可见）|
| 中景 | 下方家具顶线 或 墙面结构元素 至少1项 |
| 近景 / 特写 | 不需要 Base |

### 禁止
- 纯白无结构背景（摄影棚感，无法建立空间代入）
- 台面强行入画但与产品无关联（违反挂墙场景逻辑）

## 五、挂墙场景构图规则

### 尺度感知（核心防发散机制）

> 挂墙产品（特别是 S/M 档）极易被 AI 模型放大至占满画面。全景/中景构图必须强制包含尺度线索，形成"硬性约束"，而非依赖景别词汇的软性描述。

```
全景/中景：必须有至少1个尺度参照
参照物锚点（按约束力排序）：

  1. 微型绝对尺寸锚点（最高优先级，S/M档强制使用）
     → 标准白色电源插座（standard white power outlet）
     → 电灯开关（light switch）
     → 利用全人类已知的固定小尺寸（约86×86mm），强制AI收缩产品比例
     → 写法：a standard white power outlet visible on the wall below providing scale reference

  2. 相对高度上限（强约束）
     → 在提示词中明确规定产品整体高度不得超过下方家具的某个部件
     → 写法：the entire piece is no higher than the height of the sofa backrest below

  3. 大型空间锚点（提供背景高度参照）
     → 门框/窗框边缘局部入画，交代墙面的绝对高度

  4. 下方家具顶线（常规居家比例感）
     → 沙发背、控台顶面、床头板、书柜顶部边缘

  5. 人手局部入画
     → 仅在悬挂/调整动作镜头中使用
```

> 铁律：S档和M档产品在中景/远景中，仅写 `wide shot` 或 `one third of frame` 等景别描述不足以约束产品大小，必须叠加使用【微型绝对尺寸锚点】+【相对高度上限】。

### 留白方向

```
台面产品留白逻辑：四周 + 台面向前延伸
墙饰产品留白逻辑：上方墙面 + 下方家具与地面
```

| 比例 | 留白重点 |
|---|---|
| 16:9 | 左右两侧墙面留白，产品居中或三分线 |
| 9:16 | 上方墙面大留白，下方家具顶线收底 |
| 1:1 | 四周墙面均衡留白，产品为画面视觉中心 |

### 对抗性留白法则（防过大专项）

> 当渲染 S档（15–35cm）和 M档（35–65cm）挂墙产品处于中景/远景时，常规的 `wide shot` 会因AI"填满画面"的本能而失效。必须在提示词中使用极端留白描述，主动逼迫产品让出空间。

**必须使用的修饰词：**
- `expansive`、`vast`、`significant amount of blank space`

**明确占比描述：**
- `occupying only a tiny fraction of the wall space`
- `small and delicate cluster on the wall`
- `vast blank wall space surrounding the small [产品名称]`

### 主体占比参考（挂墙场景）

| 档位 | 全景 | 中景 | 近景 |
|---|---|---|---|
| S档 | 8%–15% | 20%–30% | 40%–55% |
| M档 | 15%–25% | 30%–45% | 55%–70% |
| L档 | 25%–40% | 45%–60% | 70%–85% |

### S/M档中远景防过大 Prompt 句式模板（强制执行）

> 生成S档和M档产品的中景/远景 image_prompt 时，必须将以下三段句式结构强制拼接入提示词。

**① 空间与留白基调**
```
...on an expansive, clean [墙面材质/颜色] wall with vast blank space surrounding the small [产品名称]...
```

**② 物理锚点植入**
```
...a standard white power outlet visible on the wall below providing scale reference...
```

**③ 相对高度压制**
```
...the entire piece is no higher than the height of the [下方家具，如 sofa backrest] below...
```

**完整拼接示例（M档四件套组合，客厅场景）：**
```
...on an expansive, clean warm white wall with vast blank space surrounding the small panel group above and on both sides, a standard white power outlet visible on the wall below providing scale reference, the entire group of four panels no higher than the height of the sofa backrest below...
```

## 六、新增商业动作（追加至R03动作库）

> 以下动作仅适用于D类，追加使用，不影响A/B/C类现有动作。

| 动作名称 | 适用子类 | 说明 |
|---|---|---|
| **悬挂归位** | D装饰型 / D功能型 | 手持产品将挂钩对准墙钉轻轻挂上，手松开后退出 |
| **水平调整** | D装饰型 / D功能型 | 双手轻微调整挂件水平角度，确认平整后松手 |
| **手持靠墙比划** | D装饰型 | 手持产品靠近墙面悬停，传递真实尺寸感知，不实际挂上 |
| **近身触摸材质** | D装饰型 | 手指轻抚产品表面雕花/纹路/焊接点，传递材质可信度 |
| **光影穿透展示** | D装饰型（镂空专属）| 固定机位，侧光穿过镂空图案在背景墙投下阴影图案，静态展示 |

**使用规则：**
- 全片不得重复同一个商业动作（继承R03现有规则）
- 悬挂归位 和 水平调整 不得在同一镜头出现
- 光影穿透展示全片限1次，且必须固定机位配合

## 七、叙事主线推导补充

> 本节内容同步写入 E2-narrative-engine.md 的D类推导规则。

```
D-装饰型 → 以情绪递进为骨架
参考方向：视觉冲击→材质工艺→挂墙场景代入→情绪高点→收官

D-功能型 → 以使用流程为骨架
参考方向：挂墙状态→功能演示→细节工艺→场景代入→收官
```

**D类叙事铁律：**
- 全片必须有至少1个挂墙场景镜头
- 手持靠墙比划动作镜头计入挂墙场景镜头
- 禁止全片只有台面特写而无挂墙场景

## 八、镂空型专项补充（D-装饰型子规则）

> 仅当产品为镂空设计时执行，其他产品跳过。

**光影穿透展示镜头规范：**
- 光源：强方向性侧光（非漫射光），角度30–45度
- 背景：纯净浅色墙面（白/米白），确保投影图案清晰可见
- 产品：固定不动，光源或相机微移产生投影轻微变化
- 时长：4–6s（属于慢节奏镜头）
- 此镜头优先级高，建议排在Shot2或Shot3位置

## 十、D类场景道具规则（替代R01 Decor规则）

> 墙面层道具与台面点缀道具均不做数量硬性限制，由产品类型与场景空间感自然决定。

**核心原则：空间感优先，产品主角地位不动摇。**

### 墙面层道具（画框/镜/挂件）

不做数量限制。部分产品本身即为Gallery wall套组，或场景天然需要多挂件组合，道具数量由产品定位与场景叙事决定。

**方向性参考（非强制）：**
- 单品陈列：墙面保持干净留白，突出产品本体
- 套组/Gallery wall：可出现多件挂件，营造组合陈列感
- 近景/特写：墙面道具退出画面，专注产品细节

### 台面点缀道具

不做数量限制，由场景空间感自然决定。

**方向性参考（非强制）：**
- 远景：空间大，可适当丰富，避免画面显空
- 中景：产品主导，道具点缀即可，不宜过多
- 近景/特写：无需道具

### 家具入画规则

| 景别 | 场景家具 |
|---|---|
| 近景/特写 | 不入画 |
| 中景 | 局部入画，出边框 |
| 远景 | 可完整出现 |

### 共同铁律
- 所有道具低饱和、低存在感，不抢产品视觉中心
- 道具服务于"有人住的真实居家感"，不是为了填充画面

## 十一、D类尺寸与场景景别限制

> 按产品最长边判定。景别限制写禁止项；产品占比下限为场景镜头强制执行项。

| 产品尺寸 | 景别限制 | 场景镜头产品占比下限 |
|---|---|---|
| 15cm以下 | 禁用中景、远景、全景 | ≥25% |
| 15–35cm（S档） | 禁用远景、全景 | ≥15% |
| 35–65cm（M档） | 禁用全景 | ≥10% |
| 65cm以上（L档） | 无限制 | 无下限 |

**执行铁律：**
- 景别限制只约束场景镜头，近景/中近景/特写不受限制，鼓励多用
- 产品占比下限是场景镜头的硬性约束，低于下限必须调整景别
- 禁止以"留白美感"为由让产品占比低于下限

---

## 十二、D类16:9横向延伸构图规则

> 适用于16:9画幅的挂墙场景中景镜头。

- 产品完整居中，是画面唯一完整主角
- 左侧墙面层道具（画框/装饰镜）出左边框，只露局部
- 右侧元素（落地灯/窗框）出右边框，只露局部
- 场景家具从边角斜切入画，不得完整出现
- 机位严格眼平，禁止俯视角度

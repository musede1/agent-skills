# narrative-structure.md 叙事节奏规则

> 本文件只负责节奏曲线、情绪强度、时长控制。
> 不负责每个镜头拍什么——镜头职责由 E2-narrative-engine.md 推导的叙事主线决定。

---

## 一、情绪节奏曲线（强制参考）

```
视觉冲击力
  ↑
  │    ● Shot1（最强冲击，快节奏）
  │
  │            ● Shot3（质感信任，慢节奏）
  │
  │                         ● Shot5（情绪最高点）
  │
  │       ● Shot2      ● Shot4          ● ShotN（极慢舒缓，留下余味）
  └──────────────────────────────────────→ 视频进度
```

**节奏控制规则：**
- 开场：快节奏，强冲击（视觉钩子）
- 中段：稳定展示，细节聚焦（建立信任）
- 结尾：情绪代入，极缓慢收官（促成转化）

> 镜头数量由用户输入决定，曲线按比例适配，不固定在6个镜头。

---

## 二、时长控制规则

### 单镜头时长范围
- 钩子镜头：2–3.5s（最快）
- 材质/细节镜头：4–6s（慢）
- 动作/场景镜头：3–5s（中速）
- 动作引导运镜镜头：3–6s（由事件节奏决定）
- 收官镜头：4–6s（最慢）

### 全片时长
- 由用户输入决定
- 所有镜头时长加总必须严格落在用户指定时长的 ±2s 范围内

### voice_over_script 时长控制
- 每个镜头 3–12 个单词
- 单词数参考值 = 镜头时长(s) × 2（舒缓语速上限）
- 语速：舒缓优雅

## 三、环境动态规则

> 产品本体锁死不动，但场景环境必须有生命感。

| 镜头用途 | 环境动态要求 |
|---|---|
| 场景陈设类 | 强制加，至少1种 |
| 钩子/全貌类 | 强制加，至少1种 |
| 材质/釉面特写类 | 可选，极克制 |
| 镂空/造型细节类 | 不加 |
| 交互类（有手） | 不加 |
| 动作引导运镜类 | 不加（事件本身已是动态核心）|

**环境动态素材池：**
- 窗纱被微风轻轻拂动一下，随后自然垂落
- 花材/植物顶端随室内气流极轻微颤动
- 窗外树影光斑在台面上缓缓移动
- 侧窗光线因云层移动产生极轻微亮度变化

---

## 四、光影变化规则

| 镜头类型 | 光影要求 |
|---|---|
| 钩子类 | 光影由暗转亮/快速扫入，变化幅度最大 |
| 场景陈设类 | 窗光有方向感，台面有轻柔阴影 |
| 材质特写类 | 侧光在弧面产生高光线随机位自然滑移 |
| 动作引导运镜类 | 光线配合事件节奏自然变化，不额外强加光影特效 |
| 收官类 | 暖调柔光持续烘托，退镜中保持稳定 |

禁止所有镜头都写"柔和均匀光线"——均匀光线=无光影=画面死板。

---

## 五、运镜速度节奏规范

- 钩子镜头：比其他镜头快1.5–2倍，用"迅速"、"快速扫入"等词
- 收官镜头：比其他镜头慢1.5倍，用"极缓慢"、"几乎静止般地"等词
- 中间镜头：正常速度，用"缓慢"、"平稳"等词
- 动作引导运镜镜头：速度由事件节奏决定，不预设快慢

> 运镜分三类：第一类机位运动、第二类机位固定、第三类动作引导运镜。
> 详细规则见 `R06-video-techniques.md`。

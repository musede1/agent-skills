# P1-positioning.md 产品定位

> 本文件是整个流程的第一阶段，所有后续阶段必须以本阶段输出为基础。
> 本阶段全部自动执行，无需人工介入。

---

## 一、标准输入格式

```
产品图：[上传图片]
站点：美国 / 欧洲 / 日本
类目：[填写]
材质：[填写]
功能属性：[填写]
尺寸：[填写，如"高16cm"]
时长：[填写，如"28秒"]
比例：16:9（默认）/ 9:16 / 1:1
竞品差异化：[填写，如"同类都是透明玻璃罐，我们是陶瓷+棋格图案"]
视频转化目标：[填写，如"主打密封性和颜值差异化"]
```

> `竞品差异化` 和 `视频转化目标` 为可选项，未填写时由AI基于产品信号自行判断。

---

## 二、执行顺序（强制）

```
Step 1：读取产品信号
Step 2：输出产品定位六项结果
Step 3：动态推导3个候选风格（公平竞争）
Step 4：动态推导3个候选叙事主线（公平竞争）
Step 5：将完整定位包传递给阶段2
```

禁止跳过任何步骤。
禁止在本阶段锁定最终风格或叙事主线，候选结果均由阶段2最终锁定。

---

## 三、产品信号读取

从产品图片或描述中提取以下信号：

| 信号维度 | 提取内容 |
|---|---|
| 材质 | 陶瓷 / 金属 / 玻璃 / 树脂 / 藤编 / 木质 / 石材 / 布艺… |
| 表面处理 | 哑光 / 高光 / 窑变 / 做旧 / 电镀 / 拉丝 / 手绘… |
| 色彩 | 主色（冷/暖/中性）/ 饱和度 / 色系 |
| 造型 | 现代几何 / 复古器形 / 有机曲线 / 极简 / 繁复 |
| 产品类型 | 摆件 / 花瓶 / 墙饰 / 容器收纳 / 功能性摆件 |
| 情绪信号 | 精致感 / 手作温度 / 自然感 / 艺术感 / 生活感 |
| 尺寸 | 用户输入的实际尺寸数值 |

## 四、产品定位六项输出（强制）

### 4.1 产品类型判定

| 类型 | 判定标准 |
|---|---|
| **A类** | 有明确"装/放/收/取/承载"功能的容器/收纳/功能性摆件（台面落点） |
| **B类** | 花瓶类，独立大类，不并入A类 |
| **C类** | 纯雕像/纯装饰摆件，无明确使用动作，以视觉陈列为主（台面落点） |
| **D类** | 墙饰类，以挂墙为主要落点；细分为D-装饰型（视觉陈列为主）和D-功能型（有明确使用动作）；详见 `R08-wall-decor-guide.md` |

> D类产品执行时，额外调用 `R08-wall-decor-guide.md`。

### 4.2 尺寸档位判定

**台面产品（A/B/C类）**，按产品整体外轮廓最长边或最高点判定：

| 档位 | 尺寸范围 | 适用产品示例 |
|---|---|---|
| **S档** | 8–20cm | 首饰托盘、钥匙碗、小香薰、小型雕塑 |
| **M档** | 20–35cm | 桌面收纳雕塑、中号花瓶、中型摆件 |
| **L档** | 35–60cm | 较大雕像、大号花瓶、台面主陈列件 |

> 详细规则见 `R04-size-profile-guide.md`。

**墙饰产品（D类）**，按产品展开后最长边判定，规则见 `R08-wall-decor-guide.md` 第三节。

### 4.3 自然归属空间

根据产品功能与情绪信号，列出2–4个最自然的归属空间。

**判定逻辑：**
- 产品本身的使用场景决定空间，不由风格文件决定
- 优先选择产品"最自然被使用/陈列"的真实生活场景
- 空间之间必须有差异性，不得全部是同一类型空间

**台面空间池（A/B/C类参考）：**
玄关 / 客厅茶几 / 书桌 / 卧室床头柜 / 梳妆台 / 厨房台面 / 厨房层板 / 餐边柜 / 浴室台面 / 阳台 / 窗台

**墙面空间池（D类参考）：**
客厅主墙 / 玄关墙面 / 卧室床头背景墙 / 餐厅墙面 / 书房墙面 / 走廊墙面 / 楼梯墙面

### 4.4 排除空间

明确列出产品不应出现的空间，防止阶段2选错场景。

### 4.5 核心使用动作

列出产品最自然的1–3个使用动作，直接影响阶段2的商业动作选择。

**参考动作库（来自 `R03-commercial-actions.md`）：**
放入 / 取出 / 归位 / 轻触调整 / 开盖 / 合盖 / 静态材质展示 / 场景收束
悬挂归位 / 水平调整 / 手持靠墙比划 / 近身触摸材质 / 光影穿透展示（D类专属）

### 4.6 A类专属：FI容量判定

**仅A类产品执行此项，B/C/D类跳过。**

```
【FI容量判定：根据产品图，该产品承载面/开口尺寸约为XX cm，
合理最大装载量为X件小物件，全片FI数量上限锁定为X件】
```

禁止跳过此步骤直接进入风格推导。

### 4.7 D类专属：子类判定

**仅D类产品执行此项，A/B/C类跳过。**

```
【D类子类判定：
D-装饰型：以视觉陈列为主，无明确使用动作
D-功能型：有明确使用动作（计时/置物/插花等）
判定结果：D-[装饰型/功能型]】
```

## 八、传递规则

- 阶段1的全部输出必须完整传递给阶段2
- 阶段2必须显式继承：产品类型 / 尺寸档位 / 自然归属空间 / 排除空间 / 核心使用动作 / 3个候选风格 / 3个候选叙事主线
- 禁止阶段2自行重新判定产品类型或空间归属
- 禁止阶段2在候选风格或候选叙事主线之外另选其他选项
